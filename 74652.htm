Utilitaire de réparation de Plesk : Mail

Exécuter l'utilitaire de réparation de Plesk avec l'aspect mail reconfigure les paramètres de messagerie de tous les domaines et boîtes mail. Les options suivantes sont spécifiques à cet aspect :

  • -restore-configuration : restaure la configuration du serveur de messagerie. Toutes les 24 heures, Plesk vérifie les modifications des fichiers de configuration du serveur de messagerie. S'il y a des modifications, Plesk crée un snapshot de la configuration. Quand l'option -restore-configuration est utilisée, les contenus des fichiers de sauvegarde actuels sont sauvegardés. Une liste des snapshots disponibles avec date de création s'affiche. Vous pouvez choisir le snapshot à restaurer. Cette option ne peut être utilisée qu'en mode interactif (Plesk pour Linux uniquement).
  • -only-domain-settings : reconfigure les paramètres de messagerie de domaines uniquement. Ne reconfigure pas les paramètres de messagerie de boîtes mail (Plesk pour Linux uniquement).
  • -dont-restore-password : reconfigure les paramètres de messagerie sans synchroniser les mots de passe des boîtes mail entre la base de données Plesk et le serveur de messagerie. Cette option doit être utilisée pour des domaines ou des boîtes mail spécifiques (Plesk pour Windows uniquement).

Vous pouvez indiquer des domaines ou des boîtes mail spécifiques afin de reconfigurer leurs paramètres de messagerie uniquement.

Exemples :

  • plesk repair mail : reconfigurez les paramètres de messagerie de tous les domaines et boîtes mail. L'utilitaire est exécuté en mode interactif. S'il détecte un problème, vous êtes informé et invité à autoriser la résolution du problème.
  • plesk repair mail exemple.com : reconfigure les paramètres de messagerie de toutes les boîtes mail appartenant au domaine exemple.com, ainsi que les paramètres de messagerie appliqués à tout le serveur.
  • plesk repair mail jeandupont@exemple.com : reconfigure les paramètres de messagerie de la boîte mail jeandupont@exemple.com.
