Ajouter et supprimer des serveurs de bases de données

Ajouter des serveurs de bases de données externes à Plesk

Pour utiliser des serveurs de bases de données externes avec votre serveur d'hébergement, les points suivants sont requis.

  1. Configurez un serveur de base de données externe :
    1. Installez MySQL, PostgreSQL ou Microsoft SQL.

      Les versions prises en charge sont répertoriées dans les Notes de mises à jour.

    2. Configurez le compte de l'administrateur de la base de données.
    3. Activez l'accès au réseau pour le serveur de bases de données.
  2. Connectez-vous à Plesk.
  3. Allez sous Outils & Paramètres > Serveurs de bases de données, puis cliquez sur Ajouter un serveur de bases de données.


  4. Indiquez les propriétés du serveur de bases de données :
    • Indiquez le moteur du serveur de bases de données dans la zone Type de serveur de bases de données. La liste contient les moteurs de bases de données pris en charge par votre clé de licence. Vous pouvez vérifier les composants pris en charge sur la page Outils & Paramètres > Plesk > Gestion des licences.

      Remarque : pour utiliser Microsoft SQL Server avec Plesk, vous devez avoir Plesk Power Pack.

      Notez qu'avant d'utiliser un moteur de serveur de bases de données, vous devez l'installer dans Outils & Paramètres > Mises à jour et à niveau > Ajouter/Supprimer des composants, sauf s'il a été installé avec Plesk.

    • Indiquez le nom d'hôte ou l'adresse IP du serveur de bases de données.
    • Indiquez le numéro du port d'écoute du serveur de bases de données. Cette option n'est possible que pour MySQL. Par défaut, les serveurs MySQL écoutent sur le port 3306. Vous pouvez laisser le champ Numéro de port vide si votre serveur de base de données MySQL écoute sur le port par défaut.

      Remarque : n'utilisez pas le port 8306 pour le serveur MySQL. Le port 8306 est utilisé par Plesk pour communiquer avec sa base de données interne.

    • Indiquez le type de base de données associée au serveur.
    • Pour que ce serveur de bases de données soit votre serveur par défaut pour héberger les bases de données des clients, cochez la case Prendre ce serveur par défaut pour MySQL. Si vous utilisez un serveur de bases de données MS SQL, cochez la case Utiliser ce serveur par défaut pour MS SQL.
    • Précisez le nom d'utilisateur et le mot de passe de l'administrateur du serveur de bases de données.
  5. Cliquez sur OK.

Si vous avez enregistré un serveur SQL, vous souhaiterez peut-être définir ses paramètres de sauvegarde. Pour configurer les paramètres de sauvegarde d'un serveur SQL distant, consultez la section Configurer les paramètres de sauvegarde de serveurs SQL distants.  

Supprimer des serveurs de bases de données de Plesk

Vous pouvez supprimer un serveur de bases de données uniquement s'il ne s'agit pas d'un serveur par défaut et s'il n'y a aucune base de données dessus.

Pour désenregistrer un serveur de base de données de Plesk :

  1. Allez sous Outils & Paramètres > Serveurs de bases de données.
  2. Cochez la case à droite du nom de l'hôte du serveur de bases de données.
  3. Cliquez sur Supprimer.
  4. Confirmez l'action puis cliquez sur OK.

  

Pour désenregistrer un serveur de bases de données incluant des bases de données ou par défaut pour les bases de données des clients d'hébergement :

  1. Supprimez les bases de données du serveur :
    1. Allez sous Outils & Paramètres > Serveurs de bases de données.
    2. Cliquez sur le nom d'hôte du serveur de bases de données à supprimer de Plesk.
    3. Cochez la case dans le coin supérieur gauche de la liste pour sélectionner toutes les bases de données.
    4. Cliquez sur Supprimer.
    5. Confirmez la suppression, puis cliquez sur OK.
  2. Définissez un autre serveur de bases de données par défaut :
    1. Cliquez sur le raccourci Serveurs de bases de données dans le chemin en haut de l'écran.
    2. Cliquez sur le nom d'hôte du serveur de bases de données à définir par défaut. Le serveur de bases de données doit être de même type (MySQL ou MS SQL) que le serveur que vous allez supprimer.
    3. Cliquez sur Préférences et cochez la case Utiliser ce serveur par défaut pour MySQL. Si vous utilisez un serveur de bases de données SQL, cochez la case Utiliser ce serveur par défaut pour MS SQL.
    4. Cliquez sur OK.
  3. Retournez à la liste des serveurs de bases de données (Outils & Paramètres > Serveurs de bases de données).
  4. Cochez la case correspondant au serveur de bases de données dont vous n'avez plus besoin.
  5. Cliquez sur Supprimer.
  6. Confirmez l'action puis cliquez sur OK.

Important : vous ne pouvez pas supprimer les bases de données d'applications Web de cette façon. Pour les supprimer, vous devez d'abord supprimer les applications Web respectives sur les sites où elles sont utilisées.
