Se connecter aux dossiers Web sur les systèmes Microsoft Windows

Pour connecter un dossier Web à votre ordinateur exécutant Microsoft Windows Vista ou Microsoft Windows 7 :

  1. Cliquez sur le bouton du menu Démarrer et sélectionnez Ordinateur.
  2. Cliquez sur Connecter un lecteur réseau.
  3. Sélectionnez la lettre du lecteur qui sera assigné au lecteur réseau et cliquez sur Se connecter à un site Web permettant de stocker des documents et des images.

  4. Allez dans Plesk et trouvez l'adresse du dossier Web requis. Connectez-vous à Plesk, allez dans Partage de fichiers, trouvez le dossier et ouvrez-le. Ensuite, cliquez sur l'icône dans le coin supérieur droit du Gestionnaire de fichiers. Le lien vers le dossier Web actuel s'affiche dans la fenêtre Lien vers ce dossier. Copier le lien dans le presse-papiers.

  5. Revenez à l'Assistant Ajout d'un Favori réseau, indiquez l'URL complète vers le dossier Web requis et cliquez sur Terminer.
  6. Dans la fenêtre qui s'ouvre, indiquez le nom d'utilisateur et le mot de passe que vous utilisez pour vous connecter à Plesk. Cochez la case Mémoriser mon mot de passe et cliquez sur OK.

Désormais, à chaque fois que vous démarrerez votre ordinateur, le dossier Web s'affichera dans votre Explorateur Windows, sous Emplacements réseau.

Pour connecter un dossier Web à votre ordinateur exécutant Microsoft Windows XP :

  1. Faites un clic-droit sur le bouton du menu Démarrer et sélectionnez Explorer.
  2. Dans le menu Outils, sélectionnez Connecter un lecteur réseau.
  3. Sélectionnez la lettre du lecteur qui sera assigné au lecteur réseau et cliquez sur Ouvrir une session de stockage en ligne ou se connecter à un serveur réseau.

  4. Cliquez sur Suivant.
  5. Vérifiez si l'option Choisissez un autre emplacement réseau est bien sélectionnée et cliquez sur Suivant.

  6. Connectez-vous à Plesk, allez dans Partage de fichiers, puis dans le dossier (par ex. : Lien vers ce dossier). Ensuite, cliquez sur l'icône dans le coin supérieur droit du Gestionnaire de fichiers. Le lien vers le dossier Web actuel s'affiche dans la fenêtre Lien vers ce dossier. Copier le lien dans le presse-papiers.

  7. Revenez à l'Assistant Ajout d'un Favori réseau, indiquez l'URL complète vers le dossier Web requis et cliquez sur Suivant.

  8. Dans la fenêtre qui s'ouvre, indiquez le nom d'utilisateur et le mot de passe que vous utilisez pour vous connecter à Plesk. Cochez la case Mémoriser mon mot de passe et cliquez sur OK.

  9. Indiquez un nom pour désigner ce dossier Web dans votre système d'exploitation et cliquez sur Suivant.

  10. Cliquez sur Terminer.

Désormais, à chaque fois que vous démarrerez votre ordinateur, le dossier Web s'affichera dans votre Explorateur Windows, sous Favoris réseau.

Remarque pour les utilisateurs d'autres systèmes d'exploitation Microsoft Windows : si vous rencontrez des problèmes de connexion à un dossier Web, vérifiez si vous avez installé toutes les mises à jour et tous les services packs de votre système d'exploitation. Si vous utilisez une version 32 bits de Windows XP, Windows Vista ou Windows 2003 Server, alors installez également le correctif disponible sous http://www.microsoft.com/downloads/details.aspx?familyid=17C36612-632E-4C04-9382-987622ED1D64.
