Logiciels de messagerie

Vous pouvez installer sur le serveur un certain nombre de packs logiciels de messagerie Web et sélectionner ceux devant être disponibles pour vos utilisateurs. Sinon, vous pouvez rediriger vos utilisateurs vers un service de messagerie Web externe. Pour ce faire, enregistrez une adresse de services de messagerie Web externe avec Plesk en ajoutant une entrée correspondante.

Tous les packs de messagerie Web installés et les services de messagerie Web externes enregistrés sont listés dans les paramètres de messagerie des packs d'hébergement et des abonnements (section Mail). Vous pouvez alors sélectionner la messagerie Web à fournir aux utilisateurs.

Logiciels de messagerie pris en charge

Les messageries Web Horde et RoundCube sont installées par défaut sur Plesk pour Linux. Seul Horde est installé sur Plesk pour Windows. En outre, Plesk pour Linux prend en charge Atmail. Quant à Plesk pour Windows, il prend également en charge les clients de messagerie suivants :

  • Client Web MailEnable
  • Client Web SmarterMail
  • Client Web du serveur de messagerie IceWarp (Merak)
  • Atmail Open

Important : si vous proposez Roundcube à vos clients, notez que les sauvegardes Plesk n'incluent pas les sauvegardes des bases de données Roundcube. Par conséquent, les données Roundcube des clients, comme les calendriers et les contacts, ne sont pas restaurées lorsque vous procédez à une restauration d'abonnements ou du serveur complet.

Services de messagerie Web externes

Pour enregistrer un service de messagerie Web externe :

  1. Allez dans Outils & Paramètres > Messagerie Web (dans le groupe Mail).
  2. Cliquez sur Enregistrer la messagerie Web.
  3. Indiquez les éléments suivants :
    1. Nom du service de messagerie Web.
    2. URL du service de messagerie Web. Indiquez une adresse commençant par le préfixe http:// ou https://.
    3. Ne décochez pas la case Disponible pour que le service de messagerie Web enregistré soit sélectionnable dans les paramètres du pack d'hébergement.
  4. Cliquez sur OK.

Pour modifier les propriétés d'un enregistrement de messagerie Web externe :

  1. Allez dans Outils & Paramètres > Messagerie Web (dans le groupe Mail).
  2. Cliquez sur le lien approprié dans la colonne Nom.
  3. Effectuez les changements requis, puis cliquez sur OK.

Pour qu'un service de messagerie Web ne puisse pas être inclus dans les packs d'hébergement :

  1. Allez dans Outils & Paramètres > Messagerie Web (dans le groupe Mail).
  2. Procédez comme suit :
    • Pour rendre un service temporairement indisponible sans le supprimer vraiment, cochez la case correspondante et cliquez sur Désactiver.

      Cela fonctionne pour les packs logiciels de messagerie Web installés et pour les liens vers les services de messagerie Web externes.

    • Pour supprimer définitivement un enregistrement de services, cochez la case correspondante et cliquez sur Supprimer.

      Ainsi, vous ne pouvez supprimer que les liens vers les services de messagerie Web externes. Les packs logiciels installés ne peuvent être supprimés qu'à l'aide du Programme d'installation de Plesk (Outils & Paramètres > Mises à jour).
