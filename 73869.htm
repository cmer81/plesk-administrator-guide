Licences pour Plesk Onyx

Nouvelles licences

Plesk Onyx étant proposé en trois éditions, il existe trois éditions des licences de Plesk Onyx : Web Admin, Web Pro et Web Host. Chaque licence est disponible pour un ou deux types de plates-formes :

  • VPS, pour toute plate-forme de virtualisation : Virtuozzo (conteneurs et VM), VMware, Xen, Hyper-V etc.
  • Dédié, peut être installé sur un VPS ou un serveur dédié.

Avec Plesk Onyx, il n'y a plus de licences Plesk pour Windows et Plesk pour Linux. Vous pouvez désormais installer la même licence soit sur un serveur Windows soit sur un serveur Linux.

Les mises à niveau et les rétrogradations sont disponibles uniquement pour des licences par location mensuelle. Les chemins d'accès vers les mises à niveaux suivantes sont disponibles :

  • Depuis Web Admin vers Web Pro ou Web Host
  • Depuis Web Pro vers Web Host

La mise à jour n'est également plus possible qu'au sein d'un type de plate-forme de licence : les licences VPS peuvent être mises à niveau vers des licences VPS alors que les licences dédiées peuvent être mises à jour seulement vers des licences dédiées.

Nouveaux add-ons

Voici les modifications dans le modèle d'add-on Plesk Onyx:

  • Il n'y a aucun add-on Windows ou Linux : les add-ons s'appliquent à toutes les licences (par ex. : PowerPack).
  • Les add-ons sont publiés dans les types VPS et Dédié, affectant leur tarif. Le type d'add-on doit correspondre au type de licence parent (VPS ou Dédié).
  • Même dans le cadre d'une licence perpétuelle (achat), les licences des add-ons sont des licences par location mensuelle.

Service de mise à jour logicielle (SUS)

Par le passé, le Service de mise à jour logicielle (Software Upgrade Services, SUS) était inclus dans les fonctions du domaine. Pour Plesk Onyx, la limite de domaines fait partie du type de licence, c'est pourquoi le SUS des licences perpétuelles (achat) doit être disponible en tant que fonction distincte : SUS 1 an pour Plesk (achat). Le SUS est obligatoire dans l'achat initial.

Compatibilité avec les anciennes licences

Plesk Onyx fonctionne avec les licences de location Plesk 10.x/11.x/12.x ou ultérieures. Ces licences bénéficient de propriétés supplémentaires activant les fonctions de Plesk Onyx en cas de mise à niveau de Plesk. Les licences payantes de versions antérieures de Plesk peuvent être mises à niveau vers Plesk Onyx si elles intègrent la fonction SUS. Sinon, vous devez acheter une nouvelle licence Plesk Onyx quand vous procédez à la mise à niveau vers Plesk Onyx.
