Presence Builder

Presence Builder est un outil qui permet aux utilisateurs de créer des sites professionnels sans la moindre connaissance en matière de balisage HTML ou de design graphique. Cet outil offre un simple éditeur visuel et un large jeu de templates pour différents sites Web. Il vous permet de créer des pages Web, d'ajouter du contenu de différents types (texte, images, vidéos, scripts, etc.) et d'éditer les paramètres de site Web tels que le nom du site, les mots-clés, icônes, etc.

Pour créer des sites Web dans Presence Builder, assurez-vous que votre abonnement d'hébergement inclut cette option. Si tel n'est pas le cas, choisissez une autre procédure ou mettez à niveau votre abonnement.

Pour créer un site Web avec Presence Builder :

  1. Allez dans Sites Web & Domaines et cliquez sur votre nom de domaine.
  2. Sous Presence Builder, cliquez sur Créer un site.
  3. Sélectionnez le thème adapté à votre site Web.
  4. Éditez le site Web :
    1. Structure : ajoutez plus de pages, supprimez les pages prédéfinies dont vous n'avez pas besoin.
    2. Contenu : remplacez le contenu prédéfini par le vôtre, ajoutez du texte, des images, vidéos, scripts et tous les autres éléments requis.
    3. Design : changez le style et la charte chromatique.
  5. Publiez le site Web.

Pour en savoir plus sur la création de sites Web avec Presence Builder, consultez la section Créer des sites Web avec Presence Builder.
