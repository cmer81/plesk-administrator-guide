Se connecter à Plesk pour la première fois

Une fois que l'installation est terminée à l'aide du programme d'installation automatique de Plesk, vous recevrez une URL de connexion unique contenant un token de session. Cliquez sur cette URL pour vous connecter à Plesk. Sélectionnez la langue de l'interface souhaitée. Ensuite, changez votre mot de passe et connectez-vous à Plesk à l'aide de votre nouveau mot de passe.

Se connecter à Plesk

Pour vous connecter à Plesk, ouvrez votre navigateur Web et saisissez l'adresse suivante :

https://<nom-d'hôte-ou-IP>:8443

La page de connexion de Plesk s'ouvre.

Saisissez votre nom d'utilisateur ainsi que votre mot de passe pour vous connecter. Par défaut, votre nom d'utilisateur est admin. Cependant, il a peut-être été modifié si Plesk a été installé par un tiers. Vous pouvez également vous connecter à Plesk à l'aide du nom d'utilisateur root (sur Linux) ou administrateur (sur Windows) et du mot de passe correspondant.

Remarque : vous pouvez désactiver la connexion à Plesk à l'aide des identifiants d'accès root ou administrator. Pour cela, ajoutez les lignes ci-après au fichier panel.ini.

Pour désactiver la connexion à Plesk avec les identifiants d'accès root ou administrateur :

[login]
systemAdmin = false
