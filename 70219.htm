Prévisualiser un site Web

Avec Plesk, vous et vos clients pouvez prévisualiser les sites pour lesquels la propagation DNS n'est pas encore terminée en les présentant comme des sous-domaines d'un autre domaine. Pour modifier les paramètres de prévisualisation des sites, allez dans Outils & Paramètres > Aperçu du site Web. Choisissez parmi l'une de ces quatre options :

  • Prévisualisation rapide par défaut. Cette option est recommandée et activée par défaut. Elle utilise l'adresse IP de Plesk à des fins de prévisualisation.
    • Authentification requise. Sélectionnez cette option pour activer la protection par mot de passe pour l'aperçu du site Web. Si cette option est activée, l'utilisateur devra fournir les identifiants d'accès de l'utilisateur système (spécifiés pendant la création de l'abonnement) afin de prévisualiser un site Web. Cette option est désactivée par défaut. Elle est disponible uniquement dans Plesk pour Windows.
  • Prévisualisation rapide sur un nom de domaine de Plesk. Si vous voulez utiliser un autre domaine hébergé dans Plesk pour l'aperçu du site Web, sélectionnez cette option. Vous devrez sélectionner l'un des domaines vous appartenant pour la prévisualisation du site. Remarque : la fonction d'aperçu sera désactivée pour le domaine sélectionné.
  • Prévisualisation rapide dans un nom de domaine externe. Si vous voulez utiliser un autre domaine hébergé en externe pour l'aperçu du site Web, sélectionnez cette option. Vous devrez indiquer le nom de domaine pour la prévisualisation du site. Vous devrez également configurer un enregistrement DNS "A" pour ce domaine. L'enregistrement doit être de type :
*.<IP_Plesk>.<nom_de_domaine>.

<nom_de_domaine> correspond au nom de domaine et <IP_Plesk> correspond à l'adresse IP de Plesk. Les points (.) pour IPv4 ou les deux-points (:) pour IPv6 sont remplacés par des tirets (-). Par exemple, si l'adresse IP de Plesk est 192.168.10.20, et si le nom de domaine que vous voulez utiliser est exemple.com, alors l'enregistrement A est de type :

*.192-168-10-20.exemple.com.

Si plusieurs adresses IP sont enregistrées dans Plesk, ajoutez une entrée par adresse.

  • Désactiver la prévisualisation rapide. Si vous voulez désactiver la prévisualisation des sites Web, sélectionnez cette option.
