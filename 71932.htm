Serveurs Web

Plesk utilise des serveurs Web pour l'hébergement Web et la gestion des sites Web. Plesk pour Linux utilise le serveur HTTP Apache (http://httpd.apache.org/) qui est complété par le serveur de proxy inversé Nginx pour de meilleures performances. Plesk pour Windows utilise le serveur HTTP IIS (http://www.iis.net/). Dans ce chapitre, vous trouverez les informations sur l'ajustement de la configuration des serveurs Web et l'optimisation de leur fonctionnement.
