Abonner des revendeurs à des packs

Si vous n'utilisez pas de solution d'automatisation de la facturation, pour commencer à servir un revendeur dans Plesk, vous devez créer le compte revendeur et l'abonner à un pack revendeur.

Pour abonner un nouveau revendeur à un pack revendeur :

  1. Allez dans Revendeurs, puis cliquez sur Ajouter un revendeur.

  2. Indiquez les informations de facturation/coordonnées du revendeur, le nom d'utilisateur et le mot de passe.
  3. Vous pouvez ajouter des notes ou une description à un compte revendeur. Cette description apparaît dans la liste des revendeurs. Vous seul pourrez voir cette description en tant que propriétaire d'un compte revendeur.
  4. Sélectionnez un pack revendeur auquel l'abonnement doit être associé.
  5. Ne cochez pas la case Procéder à la personnalisation des paramètres de l'abonnement une fois le revendeur créé. [...].
  6. Cliquez sur OK.
