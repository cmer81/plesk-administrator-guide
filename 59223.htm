Log d'actions

Vous voulez peut-être conserver une trace des actions effectuées par les divers utilisateurs dans le système. Toutes les actions seront enregistrées dans le fichier de log que vous pourrez télécharger pour le consulter. Les évènements système (actions) suivants peuvent être consignés dans le log :

  • Changements du profil de l'administrateur
  • Démarrage, redémarrage ou arrêt du service système
  • Ajout, suppression ou modification d'adresse IP
  • Modification des paramètres de connexion (période d'inactivité autorisée pour toutes les sessions utilisateur du panneau de contrôle)
  • Création, suppression du compte client ; modification des informations personnelles ou du système
  • Modification du statut du compte client (suspendu/activé)
  • Modification des préférences de l'interface du client
  • Modification du pool d'adresses IP du client
  • Ajout ou suppression d'applications Web d'un pool client
  • Atteinte de la limite de l'espace disque pour un compte client
  • Atteinte de la limite de l'utilisation du trafic pour un compte client
  • Atteinte de la limite de l'espace disque pour un site Web
  • Atteinte de la limite de l'utilisation du trafic pour un site Web
  • Création ou suppression d'un site Web, modification des paramètres
  • Modification du propriétaire d'un site Web
  • Modification du statut du site Web (suspendu/activé)
  • Mise à jour de zone DNS pour un site Web
  • Création, suppression d'un sous-domaine, modification des paramètres
  • Création, suppression d'un alias de domaine, modification des paramètres
  • Modification de la zone DNS de l'alias
  • Modification de l'allocation des ressources pour un compte client
  • Modification des droits du client pour les opérations
  • Modification de l'allocation des ressources pour un site Web
  • Connexion/déconnexion des utilisateurs à Plesk
  • Création, suppression, modification des comptes mail
  • Création, suppression des listes de diffusion, modification des paramètres
  • Configuration, suppression, modification de l'hébergement d'un site Web
  • Création, suppression, reconfiguration de comptes d'hébergement pour la redirection Web
  • Installation, désinstallation, reconfiguration d'applications Web
  • Installation, désinstallation, mise à jour de packs d'applications Web
  • Expiration ou mise à jour de la clé de licence
  • Création, suppression, mise à jour du serveur de bases de données
  • Création ou suppression de la base de données
  • Création, suppression, mise à jour du compte utilisateur de la base de données
  • Mise à jour de la GUID du client
  • Mise à jour de la GUID du domaine
  • Mise à jour ou ajout d'un composant Plesk
