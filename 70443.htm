Composant Watchdog (surveillance du système)

Le composant Watchdog est une solution garantissant que votre serveur est exempt de logiciel malveillant, que tous les services fonctionnent correctement et qu'il y a assez d'espace disque libre sur le serveur.

Watchdog peut surveiller les services suivants :

  • Serveur Web Plesk
  • Moteur PHP Plesk
  • Web Server (Apache)
  • Serveur SMTP (Postfix)
  • Serveur SMTP (QMail)
  • Serveur de messagerie POP3
  • Serveur de messagerie POP3 avec prise en charge SSL/TLS
  • Serveur Courier IMAP
  • Serveur Courier IMAP avec prise en charge SSL/TLS
  • Démon d'authentification de messagerie IMAP
  • Serveur Dovecot IMAP et POP3
  • Serveur DNS (BIND)
  • Tomcat
  • MySQL
  • PostgreSQL
  • Plesk SpamAssassin
  • Plesk Premium Antivirus
  • Antivirus Kaspersky
  • Filtre de milter Postfix Plesk
  • Serveur de proxy Web (Nginx)
  • PHP-FPM (gestionnaire de processus FastCGI)
  • Moniteur d'échec de l'authentification Fail2Ban

Il peut démarrer, arrêter et redémarrer les services qu'il surveille. Il peut être configuré pour entreprendre des actions en fonction de la stabilité d'un service pendant un certain laps de temps.

Il peut exécuter d'autres utilitaires et vous notifier quand l'utilisation de l'espace disque a atteint le volume indiqué.

À des fins de surveillance des services et de l'utilisation de l'espace disque, Watchdog utilise l'utilitaire monit. Pour en savoir plus sur l'utilitaire de surveillance, visitez le site Web pour développeur : http://www.tildeslash.com/monit/.

Watchdog peut analyser sur demande ou de manière planifiée le système de fichiers du serveur afin d'y rechercher les rootkits, backdoors (portes dérobées), exploits, chevaux de Troie et autres logiciels malveillants. Il peut vous notifier par mail des résultats de l'analyse et afficher les rapports sur le Panneau de contrôle. Il met à jour la Base de connaissances de sécurité via Internet avant chaque scan.

Watchdog utilise l'utilitaire Rootkit Hunter pour détecter sur le serveur tout logiciel malveillant. Pour en savoir plus sur Rootkit Hunter, visitez le site Web pour développeur de Rootkit Hunter : http://www.rootkit.nl.
