Disponibilité des applications pour vos clients

La liste des applications disponibles pour un client dépend de divers facteurs, tels que les paramètres du pack de services, la configuration du référentiel d'applications, etc. En outre, les fournisseurs de services peuvent interdire l'accès des applications à tous les utilisateurs de Plesk. Si vous n'adaptez pas la disponibilité des applications, vos clients verront toutes les applications depuis le Catalogue d'applications et toutes les applications que vous avez envoyées vers le Référentiel.

Pour voir la liste des applications disponibles pour un certain client, allez dans a section Applications > Toutes les applications disponibles. Avant qu'une application ne soit disponible dans la liste d'un certain client, elle passe toute une série de filtres. L'application est filtrée sur les niveaux suivants :

  1. Référentiel d'applications.
    Parallels Plesk vous permet de basculer la disponibilité des packs APS que vous avez envoyés au Référentiel. Notez que cela fonctionne uniquement pour vos propres packs : il n'y a aucune manière de contrôler la disponibilité des applications téléchargées à partir du Catalogue. Pour en savoir plus sur la gestion des applications, consultez la section Référentiel d'applications. 
  2. Pack de services.
    Parallels Plesk vous permet d'indiquer les applications à inclure dans un certain pack de services. Ce filtre affecte tous les clients avec ce pack de services. 
  3. Abonnement.
    Si vous voulez sélectionner les applications disponibles pour un client particulier, mettez à jour la liste des applications dans l'abonnement respectif.

La liste des applications résultante est disponible pour vos clients.
