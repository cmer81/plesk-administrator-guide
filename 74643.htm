Activer le mode passif pour FTP (Linux)

Pour permettre aux utilisateurs de se connecter à votre serveur Plesk pour Linux via FTP en mode passif, vous devez modifier manuellement le fichier de configuration ProFTPD et changer la politique de votre pare-feu pour autoriser les connexions FTP passives.

Connectez-vous à votre serveur SSH en tant qu'utilisateur root. Ouvrez le fichier /etc/proftpd.conf pour le modifier et ajoutez la ligne suivante quelque part dans la section <Global> :

PassivePorts 49152 65534

Le fichier sera similaire à ceci une fois que vous aurez ajouté la ligne :

Enregistrez les modifications et redémarrez le service xinetd pour que les modifications prennent effet :

/etc/init.d/xinetd restart

Pour vous assurer que les connexions FTP passives sont autorisées via votre pare-feu, ajoutez une règle de pare-feu personnalisée avec les paramètres suivants :

  • Direction : Entrant
  • Action : Autoriser
  • Ports : TCP 49152-65534
  • Sources : (tout hôte)
