Se connecter aux dossiers Web sur les systèmes Linux

Pour connecter un dossier Web en tant que lecteur réseau à votre ordinateur basé sur Linux à l'aide du gestionnaire de fichiers GNOME Nautilus :

  1. Ouvrez l'explorateur de fichiers.
  2. Dans le menu Fichiers, sélectionnez Se connecter à un serveur.
  3. Depuis le menu Type de service, sélectionnez soit WebDAV (HTTP) soit Secure WebDAV (HTTPS).

    Pour savoir quelle est l'option dont vous avez besoin, demandez à votre administrateur système.

  4. Dans le champ Serveur, saisissez le nom d'hôte ou l'adresse IP de votre serveur.
  5. Dans le champ Dossier, saisissez le chemin d'accès à votre dossier partagé.

    Pour trouver l'emplacement de votre dossier Web, connectez-vous à Plesk, allez dans Partage de fichiers, trouvez le dossier requis et ouvrez-le. Ensuite, cliquez sur l'icône dans le coin supérieur droit du Gestionnaire de fichiers. Le lien vers le dossier Web actuel s'affichera dans la fenêtre ouverte.

  6. Cliquez sur Se connecter.
  7. Le raccourci vers le partage de fichiers sera ajouté au volet Places. Cliquez sur ce raccourci.
  8. Indiquez le nom d'utilisateur et le mot de passe que vous utilisez pour vous connecter à Plesk.
  9. Sélectionnez l'option Mémoriser le mot de passe pour vous éviter de le saisir à chaque fois que vous voulez vous connecter au serveur.
  10. Cliquez sur Se connecter.

Pour vous connecter à un dossier Web à l'aide du Gestionnaire de fichiers Konqueror pour KDE :

  1. Ouvrez Konqueror et saisissez l'adresse du dossier partagé dans la barre d'adresse. Utilisez WebDAV ou WebDAVS en tant que protocole à la place de HTTP ou HTTPS.

    Pour trouver l'emplacement de votre dossier Web, connectez-vous à Plesk, allez dans Partage de fichiers, trouvez le dossier requis et ouvrez-le. Ensuite, cliquez sur l'icône dans le coin supérieur droit du Gestionnaire de fichiers. Le lien vers le dossier Web actuel s'affichera dans la fenêtre ouverte.

  2. Indiquez le nom d'utilisateur et le mot de passe que vous utilisez pour vous connecter à Plesk.
