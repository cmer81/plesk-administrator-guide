Modifier les fichiers

Une fois que vous avez obtenu l'archive du pack, vous devez extraire son contenu dans un répertoire et modifier les fichiers. Vous pouvez aussi installer tel quel le pack de skin téléchargé sur le serveur (comme décrit dans la section Installer des thèmes dans Plesk) et modifier les fichiers directement sur le serveur.

Important : si vous utilisez cette méthode, assurez-vous de définir les informations du thème comme décrit dans la section Modifier les informations du thème ci-dessous avant d'installer le thème.

Structure du pack du thème

Le tableau suivant décrit la structure et le contenu des répertoires et fichiers inclus dans un pack de thème.

Remarque : pour vous aider à déterminer les fichiers d'images ou les styles que vous devez modifier nous vous recommandons d'utiliser des outils de développement Wevb, comme Firebug pour Mozilla Firefox ou Inspect Elements pour Google Chrome. Ces outils vous permettent de sélectionner l'élément de l'interface et de voir le style appliqué, le fichier et la ligne définis dans ce fichier.

 

css

Le fichier custom.css dans ce répertoire est vide lorsque vous téléchargez le pack de thème. Aucun des styles définis dans le fichier inclus dans le thème de Plesk ne peut être modifié. Toutefois, si vous voulez redéfinir le style d'un élément de l'interface, vous pouvez ajouter une définition du style dans le fichier custom.css téléchargé et l'envoyer dans Plesk. Dans ce cas, seul le style que vous avez défini est appliqué à cet élément, les autres éléments ne sont pas modifiés.

Vous pouvez ajouter des définitions de style pour l'arrière-plan de la page, les principales polices, les couleurs du texte, des hyperliens, des bordures d'éléments d'écran, etc.

icônes

Ce répertoire contient uniquement les icônes placées dans les sous-répertoires par leur taille. Notez que la taille de l'icône est doublée pour les écrans Retina.

Si vous n'avez pas l'intention de personnaliser l'un de ces éléments, vous pouvez les supprimer de l'archive avec votre thème personnalisé. Vous pouvez conserver uniquement les éléments que vous avez modifiés ou ajoutés.

vos serveurs

Ce répertoire contient les images utilisées comme remplissage pour certains éléments de l'interface.

Certains de ces fichiers graphiques sont des feuilles Sprite qui comportent plusieurs images utilisées à différents endroits de l'interface. Pour cette raison, lorsque vous modifiez une image d'arrière-plan, vous devez vous assurer que le fichier correspondant n'est pas utilisé ailleurs. Mieux vaut créer un fichier graphique et modifier les feuilles de style au lieu de réutiliser simplement le fichier.

Si vous n'avez pas l'intention de personnaliser l'un de ces éléments, vous pouvez les supprimer de l'archive avec votre thème personnalisé. Vous pouvez conserver uniquement les éléments que vous avez modifiés ou ajoutés.

 

logos

Ce répertoire contient les images des logos de l'entreprise et du produit affichés dans Plesk. Pour changer un logo, préparez le fichier portant le même nom et remplacez-le dans le dossier.

meta.xml

Ce fichier contient les informations sur le pack du thème, comme le nom et la description.

Pour annuler des modifications et utiliser les styles ou images par défaut, supprimez simplement vos définitions CSS personnalisées du fichier custom.css et supprimez les images personnalisées des répertoires /images/ et /images/logos/ de votre thème personnalisé.

Modification des informations du thème

Pour modifier les informations de votre thème :

Ouvrez le fichier meta.xml et définissez les propriétés suivantes :

  • nom. Définissez le nom que vous utiliserez pour désigner ce thème. N'utilisez que des caractères latins et le signe moins ("-").
  • description. Saisissez une courte description du thème.
  • version. Indiquez la version du thème.
  • vendeur. Indiquez le nom de votre entreprise ou le vôtre.
