Listes noires DNS

Vous pouvez utiliser des listes noires d'abonnements gratuites et payantes avec votre serveur.

Pour activer la protection anti-spam basée sur le DNS Black Listing (DNSBL), procédez comme suit :

  1. Allez dans Outils & Paramètres > Paramètres du serveur de messagerie (groupe Mail).
  2. Cochez la case Activer la protection anti-spam basée sur la liste noire de DNS.
  3. Dans le champ Zones DNS pour le service DNSBL, saisissez le nom de l'hôte que votre serveur de messagerie doit interroger, par exemple : sbl.spamhaus.org.
  4. Cliquez sur OK.

Désormais, les mails provenant de spammeurs notoires devraient être rejetés avec le code d'erreur 550 (connexion refusée).

Important : si vous n'utilisez par le serveur de messagerie Qmail et si vous activez DNSBL, les expéditeurs avec les adresses IP de la liste noire ne pourront envoyer aucun mail, même s'ils passent l’authentification SMTP. Pour éviter ce problème, activez la soumission des messages comme décrit dans la section Configurer les paramètres de la messagerie pour l'ensemble du serveur.
