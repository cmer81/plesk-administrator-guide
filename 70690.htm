Appendice A : Propriétés des packs et des abonnements d'hébergement

Pour afficher les propriétés d'un pack d'hébergement, allez dans Packs de services > sélectionnez le pack dans l'onglet Pack d'hébergement.

Pour voir les propriétés d'un abonnement, allez sous Abonnements > sélectionnez un abonnement > cliquez sur Personnaliser sous Compte dans la barre latérale droite.

Les propriétés d'un pack et d'un abonnement d'hébergement sont groupées comme suit :

  • Ressources

    Il s'agit des ressources d'hébergement fournies avec un pack. Cela inclut la période de validité, la politique de sur-utilisation des ressources, les ressources système comme l'espace disque et le trafic, ainsi que les ressources du service comme les sites Web, sous-domaines, boîtes mail, bases de données, etc. Par exemple, les ressources des domaines définissent le nombre de domaines qu'un client peut enregistrer et gérer dans Plesk.

  • RAM, CPU, E/S disque

    Cela inclut les limites d'utilisation des ressources système par abonnement ; disponible uniquement dans le pack d'hébergement Plesk pouvant être ajouté à l'édition Web Host ou Web Pro.

  • Droits

    Cela inclut les services et privilèges fournis.

    Remarque : certains droits empêchent la synchronisation des paramètres des services suivants. Vous trouverez plus d'infos dans la section Droits.

  • Paramètres d'hébergement

    Cela inclut les paramètres du service d'hébergement fourni.

  • Paramètres PHP

    Cela inclut les paramètres PHP personnalisables.

  • Serveur Web (pack de services uniquement)

    Cela inclut les paramètres de configuration du serveur Web appliqués aux nouveaux domaines.

  • Messagerie (pack de services uniquement)

    Cela inclut les paramètres des services de messagerie fournis, y compris les limites du nombre de mails sortants.

    Remarque : pour les abonnements, les limites des mails sortants sont disponibles sous Abonnements > sélectionnez un abonnement > Changer les limites des mails sortants dans la barre d'outils.

  • DNS (pack de services uniquement)

    Indique si les zones DNS des domaines de l'abonnement doivent être maîtres ou esclaves.

    Remarque : si le privilège Gestion de la zone DNS est activé, ce paramètre n'est pas synchronisé et les abonnés peuvent configurer ce paramètre sur la base de domaines individuels.

  • Performance (pack de services uniquement)

    Cela inclut les paramètres affectant les performances de tous les services fournis avec le pack.

  • Logs & Statistiques (pack de services uniquement)

    Cela inclut les paramètres de stockage des statistiques et logs des abonnements du pack. Remarque : ces paramètres ne sont pas synchronisés entre un pack et les abonnements. Consultez la section Logs & Statistiques pour en savoir plus sur le changement de ces paramètres pour les abonnements.

  • Applications
    Permet de sélectionner les applications à la disposition des abonnés.

Visibilité des fonctions d'hébergement dans le Panneau Client

Depuis la version 10.4, Plesk masque pour les clients ces fonctions d'hébergement qui ne sont pas fournies par leur abonnement. La visibilité des éléments du GUI responsables d'une certaine fonction est déterminée par les limites de droits et ressources d'un abonnement. Notez que si vous (en tant qu'administrateur) vous connectez au Panneau Client, vous verrez les éléments du GUI, peu importe les droits du client. Par exemple, si le nombre de Domaines dans un abonnement est de 10 et si le droit Gestion des domaines est désactivé, un client ne verra pas le bouton Ajouter un domaine dans le Panneau Client. Néanmoins, ce bouton est disponible pour l'administrateur qui se connecte au Panneau Client.

Ressources sans limites numériques

Si le type de ressource est logique ou en d'autres termes, s'il peut être activé ou désactivé (par ex., prise en charge des langages de script), sa visibilité est contrôlée uniquement par un droit spécifique. Par exemple, si le droit Gestion des paramètres d'hébergement est accordé, le client peut activer la prise en charge de divers langages de scripts pour leur site. Si le droit n'est pas accordé, le client voit la liste des langues qui sont activées pour le site en mode Lecture uniquement. Les langues désactivées ne sont pas affichées dans la liste.
