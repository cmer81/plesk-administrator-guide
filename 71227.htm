Mode Sécurité renforcée

Le mode Sécurité renforcée protège les données confidentielles stockées dans la base de données Plesk. Cette fonction est activée par défaut sur les installations Plesk 11 et versions ultérieures. Si vous procédez à la mise à niveau depuis une version antérieure de Plesk, nous vous recommandons le mode Sécurité renforcée après mise à niveau.

Pour activer le mode Sécurité renforcée, allez dans Outils & Paramètres > Politique de sécurité.

Remarque : cette opération ne peut pas être annulée. Quand le mode Sécurité renforcée est appliqué, ces modifications surviennent :

  • Tous les mots de passe stockés dans la base de données Plesk sont chiffrés avec la clé secrète de Plesk. Ainsi, même si un tiers obtient un dump de la base de données Plesk, vos clients ne risquent rien.
  • Les données confidentielles (par ex. les mots de passe utilisateur) peuvent être récupérées via l'API de Plesk.
  • Les mails de récupération de mots de passe ne contiennent plus le mot de passe en texte brut. À la place, il s'agit d'un lien vers un formulaire de réinitialisation du mot de passe.

Remarque : si vous utilisez Odin Business Automation Standard (OBAS) comme solution de facturation complémentaire, le mode Sécurité renforcée doit être activé.
