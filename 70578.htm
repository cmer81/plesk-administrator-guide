Hébergement Web

La configuration de l'hébergement Web implique l'ajustement d'un certain nombre de paramètres du serveur Web et les paramètres d'autres services associés. Ainsi, sur Plesk pour Linux, vous pouvez désactiver des modules Apache inutilisés ; pour le serveur IIS, vous pouvez configurer son pool d'applications.
