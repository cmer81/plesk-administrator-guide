Outils de gestion de base de données

Les outils de gestion de base de données fournissent une interface Web destinée à l'administration des bases de données. Ces outils permettent également d'exécuter des requêtes SQL depuis cette interface utilisateur.

Plesk est doté des outils de gestion de base de données suivants :

  • PHPMyAdmin : pour gérer les bases de données MySQL.
  • phpPgAdmin, pour les bases de données PostgreSQL.
  • myLittleAdmin, pour Microsoft SQL Server.

Remarque : vous devez installer au préalable ces outils sous Outils & Paramètres > Mises à jour et à niveau > Ajouter / Supprimer des composants.

Activer l'accès direct à phpMyAdmin

Vous pouvez activer l'accès direct à phpMyAdmin et le rendre accessible via l'URL suivante :

https://<nom-hôte-ou-IP>:8443/phpmyadmin

Pour cela, ajoutez les lignes suivantes au fichier panel.ini :

[databaseManagement]
features.phpMyAdmin.loginForm.enabled = on

Pour ouvrir un outil de gestion de bases de données pour le serveur de bases de données sélectionné :

  1. Allez dans Outils & Paramètres > Serveurs de bases de données.
  2. Cliquez sur l'icône correspondant au serveur de bases de données à gérer. L'un des outils de gestion de base de données pris en charge (selon le moteur de la base de données) s'ouvre alors dans une nouvelle fenêtre du navigateur.
