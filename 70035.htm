Personnaliser la Vue Power User

La Vue Power User possède un sous-type, la Vue Personnalisée (disponible dans Outils & Paramètres > Gestion de l'interface), qui sert deux objectifs principaux :

  • Simplifier l'expérience utilisateur des administrateurs qui utilisent l'hébergement géré.

    Certains administrateurs ne s'occupent que des tâches d'administration basiques (surveillance de services système, administration des comptes utilisateur, etc.). Ils laissent les tâches plus complexes, comme la configuration des services et du serveur, au service d'assistance d'un fournisseur de services. Ce type d'utilisateur ne veut avoir que les outils dont il a vraiment besoin et masquer les autres outils.

  • Augmenter la sécurité et la convivialité de Plesk.

    Les administrateurs peuvent volontairement révoquer certains de leurs droits pour masquer les outils dont ils n'ont pas besoin au quotidien. Ils peuvent également revenir à la vue Power User qui inclut toutes les fonctionnalités s'ils ont besoin de paramétrer plus finement le système (par exemple, activer les sauvegardes du serveur).

Si vous allez dans Outils & Paramètres > Paramètres de la Vue personnalisée (onglet Outils d'administration), vous pouvez sélectionner les outils que l'administrateur verra dans cette vue. Les paramètres de cette vue peuvent être indisponibles si le fournisseur de services qui vous a donné l'accès à Plesk a décidé de verrouiller la Vue personnalisée.

Verrouillage de la Vue personnalisée et masquage des paramètres de la Vue personnalisée

Verrouiller la Vue personnalisée signifie limiter l'éventail des fonctions disponibles de Plesk pour l'administrateur et désactiver tout changement dans la liste des fonctions. Résultat : lorsque la Vue personnalisée est verrouillée, il est impossible de basculer vers une autre vue de l'interface (ou API XML) ou de changer les paramètres de la Vue personnalisée. En général, si vous êtes un fournisseur de services, vous avez la possibilité de mettre certains outils à la disposition des administrateurs et ainsi de diviser en deux l'administration de Plesk :

  • Opérations quotidiennes. Ces opérations sont exécutées par l'administrateur de Plesk, la personne qui a acheté l'hébergement Web.
  • Configuration et maintenance complexes. Ces opérations sont exécutées par votre équipe d'assistance. De telles opérations peuvent inclure la configuration d'un réseau, DNS, serveur Web, etc.

Si un administrateur de Plesk a besoin d'une certaine fonction et n'arrive pas à la trouver, votre équipe d'assistance active cette fonction en déverrouillant la Vue personnalisée, en modifiant les paramètres de la vue et en reverrouillant cette même vue.

La Vue personnalisée est verrouillée uniquement via un appel de la ligne de commande de l'utilitaire poweruser :

poweruser --on -simple true -lock true

Le verrouillage est supprimé en appelant poweruser --on -lock false.

Vue personnalisée et abonnements propres de l'administrateur

Quelle est la spécificité de cette vue ? Elle vous permet d'adapter instantanément les droits, paramètres d'hébergement, paramètres PHP et autres paramètres d'abonnement sur tous les abonnements que vous avez créés dans cette vue. Cette possibilité existe car chaque abonnement que vous créez dans cette vue vient du pack de services Personnalisée artificiel qui n'est pas visible dans la liste des packs. Les paramètres de ce pack sont disponibles dans Outils & Paramètres > Paramètres de la Vue personnalisée. Lorsque vous modifiez les paramètres, ces changements (si possible) sont automatiquement appliqués à tous les abonnements sous le pack Personnalisé.

Autre point qui mérite votre attention : le pack Personnalisé inclut un droit spécial, la possibilité de créer, de supprimer et de basculer entre les abonnements. Si ce droit n'est pas sélectionné dans l'interface utilisateur, il est impossible de créer des abonnements dans la Vue personnalisée.

Si vous devez affiner les paramètres de la Vue personnalisée via la ligne de commande, utilisez l'utilitaire admin. Pour en savoir plus sur les options de l'utilitaire, reportez-vous à Plesk Onyx pour Linux (Windows) : Référence pour les utilitaires de ligne de commande.
