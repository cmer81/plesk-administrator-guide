Télécharger des fichiers de sauvegarde stockés sur le serveur

Pour télécharger un fichier de sauvegarde depuis le stockage de Plesk :

  1. Choisissez le nom du fichier de sauvegarde correspondant dans Outils & Paramètres > Gestionnaire de sauvegardes.
  2. Cliquez sur la flèche pour télécharger le fichier et indiquez l'emplacement où enregistrer le fichier.

Pour renforcer la sécurité de la sauvegarde, nous vous conseillons de la protéger par un mot de passe. Dans ces conditions, aucun attaquant ne pourra obtenir de données confidentielles si la sécurité de votre stockage de sauvegarde venait à être compromise.

Remarque : si vous oubliez votre mot de passe, il ne pourra pas être restauré ; par conséquent, nous vous conseillons vivement de garder une liste de vos mots de passe et noms de fichiers de sauvegarde correspondants dans un endroit sûr.

Définir un emplacement pour les fichiers de sauvegarde temporaires

Lorsqu'un utilisateur télécharge une sauvegarde depuis le serveur, Plesk crée un fichier d'archive temporaire avec la sauvegarde sur le serveur. Par défaut, ce fichier est situé dans le répertoire /tmp. S'il n'y a pas suffisamment d'espace disque pour le stockage du fichier temporaire sur la partition correspondante, il est impossible de télécharger le fichier de sauvegarde. Pour éviter ce type de situation, vous pouvez modifier l'emplacement des fichiers de sauvegarde temporaires sur le serveur Plesk. Pour cela, modifiez la variable DUMP_TMP_D dans le fichier de configuration /etc/psa/psa.conf.
