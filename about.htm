À propos de Plesk

Plesk est conçu pour aider les experts IT à gérer les services Web, DNS, mail et autres via une interface complète et conviviale. Plesk est un panneau de contrôle d'hébergement, c'est-à-dire un intermédiaire entre les services système et les utilisateurs. Par exemple, lorsqu'un utilisateur crée un site Web via l'interface utilisateur de Plesk, Plesk propage cette requête vers un serveur Web (Apache ou IIS), qui ajoute alors un nouvel hôte virtuel dans le système. Cette méthode d'administration de tous les services système à partir d'une même interface Web réduit les frais de maintenance et donne plus de contrôle et de souplesse aux administrateurs.

Comment puis-je utiliser Plesk ?

Plesk est un outil clé pour les hébergeurs (HSP), c'est-à-dire les entreprises vendant des comptes d'hébergement mutualisé et dédié. Plesk étant installé sur un serveur, il leur permet d'organiser les ressources serveur en packs, permettant aux hébergeurs de proposer ces packs à leurs clients, entreprises et particuliers, ayant besoin d'une présence Web mais ne disposant pas de l'infrastructure IT requise. Pour en savoir plus sur les profils ciblés par Plesk, consultez la section À propos des utilisateurs de Plesk.

Comment personnaliser Plesk pour qu'il réponde à mes besoins ?

Chaque groupe d'utilisateurs dispose de sa propre interface personnalisée en fonction de ses besoins. Ainsi, les hébergeurs disposent des outils permettant de proposer des services d'hébergement, dont une solution de facturation qui automatise cette tâche. En revanche, les entreprises qui utilisent Plesk pour gérer leur propre infrastructure Web n'ont pas besoin des fonction de vente d'hébergement dans leur interface. À la place, elles peuvent réaliser des opérations de gestion du serveur, de type restauration système, configuration serveur Web, etc. Pour en savoir plus sur l'interface utilisateur de Plesk, consultez la section Interface de Plesk.

Dans la suite de ce chapitre, nous expliquerons comment différents groupes d'utilisateurs se servent de Plesk pour profiter de l'ensemble de ses fonctions.
