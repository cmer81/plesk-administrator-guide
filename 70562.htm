Interface de Plesk

Les outils de Plesk permettant d'exécuter les tâches de gestion des comptes et des serveurs sont divisés en deux interfaces Web appelées panneaux : le Panneau d'administration du serveur, le Panneau Revendeur et le Panneau Client.

  • Le Panneau d'administration du serveur permet de gérer les comptes et la maintenance du serveur. Les opérations d'hébergement Web sont également disponibles dans ce panneau.
  • Le Panneau Revendeur est une version allégée du Panneau d'administration du serveur destinée aux revendeurs.
  • Le Panneau Client est focalisé sur les opérations d'hébergement Web et permet de créer et de gérer les sites Web, boîtes mail, etc.

Les outils visant à servir les clients sont disponibles uniquement dans le Panneau d'administration du serveur et dans le Panneau Revendeur, tandis que les outils visant à servir les revendeurs sont disponibles uniquement dans le Panneau d'administration du serveur. Pour en savoir plus sur les services aux clients et aux revendeurs, consultez la section Clients et revendeurs. Pour en savoir plus sur les interfaces destinées à la gestion des revendeurs et des clients, consultez la section Naviguer rapidement parmi les objets Plesk.

Vous trouverez une courte description de chaque type de panneau ci-dessous.

  

Panneau d'administration du serveur

Le Panneau d'administration du serveur est l'instrument principal des hébergeurs qui leur permet de servir leurs clients et d'assurer la maintenance du serveur. Par exemple, l'administrateur crée des packs d'hébergement et des comptes client, configure les paramètres globaux du serveur des services système, etc. Les opérations d'hébergement Web sont également disponibles dans ce panneau.

Les deux principaux groupes d'administrateurs de Plesk (c'est-à-dire les revendeurs et les professionnels du Web) ont leurs propres scénarios d'utilisation de Plesk. Pour répondre aux besoins de chaque groupe, le Panneau d'administration du serveur propose deux vues. Ces vues regroupent une combinaison d'outils ciblés selon les utilisateurs : la vue Fournisseur de services et la vue Power User.

 

Vue Fournisseur de services.
Cette vue est pratique pour les hébergeurs, car elle est conçue pour vendre des services d'hébergement. Elle inclut toutes les fonctions nécessaires pour créer et gérer les comptes client, abonnements et packs de services.

Vue Power User.

Cette vue est la plus appropriée pour les professionnels du Web et les admins Web, ceux qui ne servent pas de clients et utilisent Plesk à des fins personnelles, par exemple, pour développer des sites Web ou maintenir le portail d'une entreprise.

Cette vue inclut l'administration du serveur et la gestion des services d'hébergement. Les fonctions de gestion du serveur sont accessibles via la section Outils & Paramètres dans le volet de navigation.

Toutefois, cette vue n'offre pas la possibilité de gérer les packs d'hébergement, abonnements, revendeurs et clients. Pour cela, utilisez la vue Fournisseur de services.

Changement de vue dans Plesk.

Pour passer rapidement d'une vue à l'autre, cliquez sur Changer la vue en bas à gauche de l'écran et sélectionnez une autre vue. Vous pouvez changer la vue de Plesk depuis Outils & Paramètres > Gestion de l'interface.

Personnalisation de la Vue Power User.

La Vue Power User peut être personnalisée pour les besoins d'un administrateur si vous sélectionnez Vue personnalisée depuis Outils & Paramètres > Gestion de l'interface. Pour en savoir plus sur la personnalisation, consultez la section Personnaliser la vue Power User.

 

Panneau Revendeur

Ce panneau est similaire au Panneau d'administration du serveur mais ne fournit pas les outils qui servent les revendeurs. De plus, ses capacités de gestion du serveur sont limitées. Vous pouvez l'utiliser avec les deux vues, comme le Panneau d'administration du serveur.

Panneau Client

La tâche principale du Panneau Client consiste à gérer les services d'hébergement. Les clients utilisent ce panneau de contrôle pour ajouter des domaines et boîtes mail, gérer le contenu du site Web, etc. Les administrateurs de Plesk et les revendeurs peuvent utiliser ce panneau pour créer des abonnements d'hébergement et accéder aux comptes client.

Pour en savoir plus sur l'hébergement Web, consultez la section Gérer des sites Web.

 

Presence Builder

En plus des fonctions standard de Plesk, la suite Plesk inclut un composant facultatif qui améliore significativement les fonctionnalités de Plesk : Presence Builder. Il s'agit d'un outil de création de sites Web qui permet aux clients de créer des sites Web professionnels en un temps record, sur la base de plus de 100 templates de sites. Presence Builder est étroitement intégré à Plesk et dispose de sa propre interface utilisateur. Pour en savoir plus sur cet outil, consultez la section Créer des sites Web avec Presence Builder.
