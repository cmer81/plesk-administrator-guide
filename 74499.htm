Déplacer une base de données vers un autre abonnement

Plesk vous permet de déplacer des bases de données d'un abonnement à l'autre. Cette option est pratique notamment si vous déplacez un site Web qui fonctionne avec une base de données. Vous pouvez déplacer uniquement les bases de données qui ne sont pas utilisées par les applications APS depuis le Catalogue d'applications.

Quand vous déplacez une base de données vers un autre abonnement, les utilisateurs de ces bases de données sont également déplacés. Toutefois, si un compte utilisateur est universel et dispose de l'accès aux autres bases de données, alors il n'est pas déplacé avec la base de données.

Remarque : si l'administrateur de Plesk a déplacé une base de données vers un abonnement qui appartient à un autre utilisateur de Plesk, alors l'utilisateur précédent ne pourra plus accéder à la base de données via phpMyAdmin. Cela s'applique également aux bases de données phpPgAdmin et myLittleAdmin.

Pour déplacer une base de données vers un autre abonnement :

  1. Allez dans Sites Web & Domaines > Bases de données > Déplacer vers l'abonnement dans le volet d'outils de la base de données.
  2. Sélectionnez un abonnement dans la liste ou tapez les premières lettres de son nom et sélectionnez-le.
  3. Cliquez sur OK.
