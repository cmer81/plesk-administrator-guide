Catalogue des extensions

Remarque : cette section décrit le Catalogue des extensions disponible dans Plesk Onyx 17.5 et dans les versions ultérieures. Si vous utilisez une version antérieure à Plesk Onyx, consultez la section Extensions Plesk à la place.

Les extensions Plesk sont des modules indépendants qui ajoutent de nouvelles fonctions et fonctionnalités à Plesk. Elles sont créées aussi bien par Plesk que par de nombreux partenaires de Plesk (en fait, n’importe qui peut créer et publier sa propre extension Plesk dans le Catalogue des extensions en utilisant la documentation en libre accès). Vous pouvez installer autant d’extensions que vous le souhaitez sur votre serveur Plesk.

Est-ce que j'ai besoin dextensions ?

  • Les extensions vous permettent de sélectionner les fonctions et fonctionnalités, en payant pour celles qui ont de l’intérêt dans votre situation spécifique.
  • Sur les serveurs d’hébergement partagés, vous pouvez transformer les extensions en source de revenus supplémentaires en facturant vos clients pour l’utilisation des fonctionnalités assurées par les extensions.

Les extensions Plesk peuvent être installées uniquement par l'administrateur Plesk. Certaines extensions ajoutent des fonctions qui peuvent être utilisées uniquement par l'administrateur Plesk et d'autres ajoutent également des caractéristiques aux comptes revendeur et clients.

 

Les extensions sont-elles sécurisées ?

Toutes les extensions qui se trouvent dans le Catalogue des extensions Plesk, qu’elles soient publiées par Plesk, un partenaire de Plesk, ou un tiers, sont rigoureusement vérifiées et testées. Vous pouvez toujours compter sur elles pour être en sécurité à 100%. Toutefois, vous devez être prudent avec les extensions qui viennent d'autres sources. Plesk vous conseille d'installer uniquement les extensions qui viennent de sources dans lesquelles vous avez confiance.

 

Les extensions sont-elles gratuites ?

En bref, certaines le sont et d'autres pas. L'auteur de l’extension détermine si l’extension est gratuite ou payante, choisit le modèle de licence et détermine le prix.

Il existe plusieurs modèles de licence d’extension disponibles :

  • Gratuit. Comme son nom l’indique, l’extension est gratuite. N'importe qui peut utiliser toutes les fonctionnalités sans payer.
  • Freemium. Cela signifie que certaines fonctionnalités de l’extension peuvent être utilisées gratuitement, et certaines peuvent être utilisées uniquement après l’achat d’une licence. Par exemple, une extension freemium peut scanner le serveur pour rechercher les vulnérabilités en version sans licence, et les réparer automatiquement avec la version sous licence.
  • Offre simple. Une licence payante est nécessaire pour faire fonctionner l’extension. Toutefois, le prix de la licence est toujours le même.
  • Offres multiples. Une licence payante est nécessaire pour faire fonctionner l’extension, et des licences de plusieurs niveaux sont disponibles à différents tarifs. Par exemple, des licences permettant l’utilisation de l’extension pour 10, 50 ou un nombre illimité de domaines sont disponibles.
  • Pay-as-you-go. Avec ce modèle de licence, vous payez pour la quantité de ressources que vous consommez au lieu d’un montant fixe.
 

Catalogue des extensions

Le Catalogue des extensions est un répertoire en ligne d’extensions Plesk accessibles publiquement à l'adresse https://ext.plesk.com/. Vous pouvez aussi accéder au Catalogue des extensions à partir de Plesk en cliquant sur Extensions dans le volet de navigation.

Vous aboutirez à l’onglet Catalogue des extensions. De là, vous pouvez accéder à un grand nombre d’extensions qui ajoute tous types de fonctions et fonctionnalités à Plesk. Les extensions sont triées en catégories, comme "Backup", "DNS" ou "Mail". Chaque extension appartient à une ou plusieurs catégories.

Les extensions présentées s’affichent sur la landing page. Toutefois, beaucoup de catégories contiennent plus d’extensions qu'il n’est possible d’en afficher. Pour trouver une extension spécifique, utilisez la fonction de recherche (la fenêtre de recherche est située dans le coin supérieur droit).

Si vous ne connaissez pas le nom de l’extension que vous recherchez, cherchez le menu "Sélectionner la catégorie" et sélectionnez la catégorie à laquelle elle est, à votre avis, le plus susceptible d'appartenir.

Vous n'avez pas trouvé d’extension à votre goût ? Vous avez deux options :

  • Demandez que l’extension soit créée. Cliquez sur Extensions et cherchez la section "Demander une extension" dans le coin inférieur droit. Cliquez sur le lien "Dites-nous" et vous serez amené à la plage Plesk UserVoice. Là, vous pouvez laisser une requête décrivant l’extension que vous recherchez. Les autres utilisateurs Plesk peuvent voter pour votre demande, et plus elle reçoit de voix, plus il y a de chance que Plesk crée et publie l’extension que vous demandez. Toutefois, veuillez noter que, quel que soit le nombre de votes, la décision finale de créer l’extension est réservée à Plesk. Plesk ne fournit pas non plus de garanties ou ETA.
  • Créer vous-même l’extension. Lisez la documentation en libre accès pour apprendre comment créer des extensions Plesk.
 

En savoir plus sur une extension

Pour en savoir plus sur une extension, trouvez-la dans le Catalogue des extensions puis cliquez sur le nom de l’extension.

Sur la page de l’extension, toutes sortes d'informations sont disponibles sur l’extension. Prêtez attention aux informations suivantes :

  • Version. La version la plus récente de l’extension actuellement disponible dans le Catalogue des extensions. Si vous avez une version plus ancienne de l’extension installée, c’est une bonne idée de la mettre à jour.
  • Description. Vous pouvez trouver ici un bref sommaire des fonctions et fonctionnalités que l’extension ajoute à Plesk et les avantages qu’elle procure.
  • Changements. Il s'agit d'un log de modifications qui détaille les correctifs de bogues et les améliorations fournis par les mises à jour de l’extension.

Beaucoup d’extensions fournissent également un ou plusieurs liens utiles :

  • Site Web du fournisseur. Le lien vous amène au site Web du vendeur de l’extension (la personne ou l’organisation qui a créé l’extension).
  • Aide & documentation. Ce lien vous amène aux documents d'aide accessibles au public pour l’extension (par exemple documentation en ligne ou tutoriels vidéo).
  • Assistance. Ce lien vous amène à une page Web où vous pouvez obtenir de l'aide pour l’extension (par exemple un forum ou un bureau d'aide en ligne).
 

Installation des extensions

Alors que la plupart des extensions Plesk peuvent être installées et fonctionnent aussi bien sur les serveurs Linux et Windows, certaines peuvent fonctionner uniquement sur l’un ou sur l'autre. D'autres extensions peuvent perdre certaines de leurs fonctions et fonctionnalités si elles sont installées sur un système d’exploitation spécifique.

De plus, chaque extension nécessite une version Plesk minimum sur laquelle elle peut fonctionner, et ne fonctionne pas si elle est installée sur un serveur disposant d'une version Plesk antérieure. Par exemple, une extension qui nécessite "Plesk 12.0 +" fonctionnera sur Plesk 12.0, Plesk 12.5 et Plesk Onyx, mais pas sur Plesk 10.4.

Lorsque vous installez des extensions à partir du Catalogue des extensions, les extensions pour lesquelles vous ne répondez pas aux configurations requises ne s'affichent pas. Lorsque vous installez une extension à partir d’un fichier, vous devez vous assurer que la configuration requise pour l’installation est présente. Pour connaître la configuration requise pour l’installation d'une extension, vérifiez dans le Catalogue des extensions ou contactez le vendeur de l’extension.

Pour installer une extension à partir du Catalogue des extensions, suivez les étapes suivantes :

  1. Cliquez sur Extensions.
  2. Cherchez l’extension que vous voulez installer.
  3. Cliquez sur Installer pour installer l'extension. Si vous voyez le bouton "Acheter", cela signifie que l’extension nécessite une licence payante pour l'activation de tout ou partie de ses fonctionnalités.

Pour installer une extension à partir d’un fichier, suivez les étapes suivantes :

  1. Cliquez sur Extensions et allez sous Mes extensions.
  2. Cliquez sur Envoyer l'extension. Sélectionnez le fichier contenant le pack d’extension que vous souhaitez installer, puis cliquez sur Ok.

Attention : I’installation d’extensions à partir de sources autres que le Catalogue des extensions peut être risquée et peut permettre à un tiers de nuire à votre serveur. N’installez jamais d’extensions de sources dans lesquelles vous n’avez pas confiance.

Extensions payantes

Si vous installez une extension payante, vous pouvez avoir besoin d'acheter une licence pour bénéficier de la totalité ou de certaines des fonctionnalités et fonctions fournies. Vous pouvez acheter une licence sur la page de l’extension en cliquant sur le bouton Acheter.

Si vous n’êtes pas sûr que l’extension nécessite une licence ou pas, lisez la documentation de l’extension ou contactez le vendeur de l’extension.

 

Utilisation des extensions

Bon, vous avez installé une extension - et maintenant ? Cela dépend de l’extension en question. Certaines extensions Plesk peuvent fonctionner directement, et certaines nécessitent d’être configurées avant de pouvoir être utilisées. La façon dont vous faites fonctionner une extension et bénéficiez des fonctions et fonctionnalités qu’elle offre varie d'une extension à l’autre.

  • Certaines extensions (par exemple "Skins et charte chromatique", "Plesk Migrator") n'ajoutent pas de commandes supplémentaires à l’interface Plesk. Pour bénéficier de leurs fonctions et fonctionnalités, vous devez cliquer sur Extensions dans le volet de navigation, aller sous Mes extensions et cliquer sur Aller à l’extension.
  • Certaines extensions (par exemple "Docker") ajoutent des commandes et des boutons que seul l'administrateur peut voir.
  • Certaines extensions (par exemple "Ruby", "Node.js") ajoutent des commandes et des boutons qui sont visibles par l’administrateur Plesk, les revendeurs et les clients.
  • Certaines extensions (par exemple "Plesk Cgroups Manager") s’intègrent aux abonnements et packs de service, en ajoutant des commandes supplémentaires qui peuvent être utilisées pour gérer la disponibilité des fonctions et fonctionnalités fournies par l’extension aux différents comptes revendeur et client.

Si vous n’êtes pas sûr de la façon dont s’utilise l’extension que vous avez installée, lisez la documentation de l’extension ou contactez le vendeur de l’extension. Si vous avez installé une extension, mais êtes dans l’impossibilité de faire usage d’une ou plusieurs des fonctionnalités annoncées, assurez-vous que vous avez la version la plus récente de l’extension et mettez-la à jour, si nécessaire.

 

Configuration des extensions

Beaucoup d’extensions disposent de plusieurs paramètres configurables. Les paramètres exacts varient beaucoup d’une extension à l'autre. En général, ils commandent la façon dont l’extension se comporte, mais parfois ils représentent la totalité de l’interface de l’extension (c’est-à-dire que vous avez besoin d'aller sur l’écran de configuration de l’extension pour la faire fonctionner).

Pour modifier les paramètres d’une extension installée, suivez les étapes suivantes :

  1. Cliquez sur Extensions dans le volet de navigation et allez sous Mes extensions.
  2. Sélectionnez l’extension que vous voulez configurer puis cliquez sur Aller à l’extension.

Si vous n’êtes pas sûr de l’effet produit par une option ou un paramètre spécifique, lisez la documentation de l’extension ou contactez le vendeur de l’extension.

 

Mise à jour des extensions

Après la publication d’une extension, son vendeur peut sortir des mises à jour qui apportent de nouvelles fonctionnalités, des améliorations de sécurité, des correctifs de bogues, etc. Les mises à jour des extensions sont toujours gratuites, et Plesk conseille de garder à jour toutes les extensions installées.

Pour voir s’il existe des mises à jour disponibles pour les extensions installées, cliquez sur Extensions dans le volet de navigation. Lorsque des mises à jour sont disponibles pour une ou plusieurs extensions installées, une notification s'affiche dans l’onglet "Mises à jour". Le nombre présent dans le carré rouge indique le nombre d’extensions pour lesquelles des mises à jour sont disponibles.

Pour voir quelles extensions peuvent être mises à jour, allez sous Mises à jour.

Les informations importantes ici sont le nom de l’extension, la version actuellement installée et la version disponible. Vous pouvez cliquer sur "Voir le log de modification" pour voir la liste des correctifs de bogues et améliorations qui seront fournis par la mise à jour.

Lorsqu’une mise à jour est publiée pour une extension, si la mise à jour automatique est activée sur le serveur, l’extension est mise à jour dans la semaine. Si vous ne souhaitez pas attendre, vous pouvez mettre à jour l’extension manuellement.

Mise à jour automatique des extensions

Par défaut, la mise à jour automatique des extensions est activée. Cela signifie qu’une fois par semaine, Plesk contrôle la présence de mises à jour pour toutes les extensions installées et installe automatiquement toutes les mises à jour trouvées. Si vous avez besoin de mettre à jour une extension immédiatement, vérifiez manuellement la présence de la mise à jour, comme décrit plus loin dans cette section.

Si vous craignez que l’installation d'une mise à jour puisse nuire au bon fonctionnement d'une extension, ou si vous souhaitez commencer par tester l’extension mise à jour dans un environnement de développement, il se peut que vous souhaitiez désactiver la mise à jour automatique. Pour ce faire, cliquez sur Extensions dans le volet de navigation, allez sous Mises à jour, puis cliquez sur le curseur vert "Mise à jour automatique".

Si vous changez d'avis, cliquez sur le curseur gris pour réactiver la mise à jour automatique.

Mise à jour manuelle des extensions

Pour mettre à jour manuellement une extension installée, suivez les étapes suivantes :

  1. Cliquez sur Extensions dans le volet de navigation et allez sous Mises à jour.
  2. Cliquez sur Vérifier pour vérifier la disponibilité de mises à jour pour toutes les extensions installées.
  3. Sélectionnez les extensions que vous voulez mettre à jour et cliquez sur Mise à jour.
Mise à jour d’extensions non trouvées dans le Catalogue des extensions

Notez que les procédures décrites ci-dessus conviennent uniquement aux extensions qui se trouvent dans le Catalogue des extensions. La seule façon de mettre à jour une extension qui ne se trouve pas dans le Catalogue des Extensions est de télécharger une version plus récente sous forme de fichier.

 

Désactivation des extensions

Si vous souhaitez arrêter temporairement d’utiliser une extension installée sans la supprimer, vous pouvez la désactiver. Vous pouvez souhaiter désactiver une extension, par exemple, dans une des situations suivantes :

  • Vous pensez qu’une extension installée peut poser des problèmes.
  • Vous souhaitez empêcher temporairement vos clients d’utiliser une extension installée.
  • Vous souhaitez supprimer une extension et vous assurer que cela ne vous affectera pas de façon négative.

La désactivation d’une extension préserve sa configuration et ses données, mais supprime toutes les commandes qui lui sont associées de l’interface Plesk. Les conséquences exactes de la désactivation d'une extension diffèrent selon l’extension en question.

Pour désactiver une extension installée, suivez les étapes suivantes :

  1. Cliquez sur Extensions dans le volet de navigation et allez sous Mes extensions.
  2. Sélectionnez l’extension que vous voulez désactiver, puis cliquez sur Désactiver.

Pour activer une extension désactivée, suivez les étapes suivantes :

  1. Cliquez sur Extensions dans le volet de navigation et allez sous Mes extensions.
  2. Sélectionnez l’extension que vous voulez activer, puis cliquez sur Activer.
 

Supprimer les extensions

Si vous ne voulez plus utiliser une extension installée, vous pouvez la supprimer. Lorsque vous supprimez une extension installée, toutes les données de l'extension sont supprimées (d'une extension à l'autre, les "données" varient, cf. les exemples ci-dessous). De plus, quand une extension est supprimée, tous les contrôles associés à cette extension sont supprimés de l'interface Plesk.

Avant de supprimer une extension, Plesk vous recommande de la désactiver pendant quelques jours. Si vous ne constatez aucun effet indésirable, vous pouvez supprimer cette extension en toute sécurité.

Attention : quand une extension est supprimée, toutes les données de cette extension sont perdues définitivement.

Pour supprimer une extension installée, suivez ces étapes :

  1. Dans le volet de navigation, cliquez sur Extensions, puis allez dans Mes extensions.
  2. Sélectionnez l'extension que vous voulez supprimer, puis cliquez sur Supprimer.

Les conséquences exactes liées à la suppression d'une extension varient en fonction de l'extension. Par exemple :

  • Si vous supprimez l'extension WordPress Toolkit, tous les contrôles associés disparaissent de l'interface de Plesk. Toutefois, les instances WordPress créées à l'aide de cette extension ne sont pas supprimées (en effet, il ne s'agit pas en soi de données de l'extension). Ces instances continuent de fonctionner normalement. Les modifications appliquées par l'extension aux fichiers de configuration du serveur Web et de WordPress (p. ex., .htaccess) ne seront pas annulées.
  • Si vous supprimez l'extension "Let's Encrypt", vous perdez la possibilité d'émettre des certificats Let's Encrypt via l'interface de Plesk. Les certificats déjà émis et installés continueront de fonctionner normalement, toutefois, ils ne pourront pas être renouvelés automatiquement quand ils arriveront à expiration.
  • Si vous supprimez l'extension "Node.js", les données d'application chargées restent, toutefois les applications elles-mêmes cesseront de fonctionner.
