Utiliser un stockage FTP

Si vous allez utiliser un serveur FTP pour stocker les fichiers de sauvegarde, vous devez indiquer ses paramètres dans Outils & Paramètres > Gestionnaire de sauvegardes > Paramètres du stockage FTP. Cliquez sur Utiliser le stockage FTP et complétez ces informations :

  • Adresse IP du serveur FTP ou nom d'hôte.
  • Répertoire sur le serveur dans lequel vous voulez stocker les fichiers de sauvegarde.
  • Nom d'utilisateur et mot de passe à saisir pour accéder au compte FTP.
  • Utilisation de FTP en mode passif. Si la connexion au serveur FTP indiqué échoue, sélectionnez cette option et réessayez. Plesk teste la connexion lorsque vous cliquez sur OK sur l'écran des Paramètres du stockage FTP.
  • Utilisation de FTP en mode passif. Si le serveur FTP prend en charge ce protocole, vous pouvez l'utiliser pour améliorer la sécurité.
  • Mot de passe que Plesk utilisera pour protéger la sauvegarde.

    Remarque : la protection par mot de passe sécurise uniquement les mots de passe des utilisateurs inclus dans les sauvegardes. Cette protection ne s'applique pas aux fichiers utilisateur ou à tout autre contenu.

Comment copier les sauvegardes vers le stockage du serveur

Pour copier une sauvegarde depuis le stockage FTP vers le stockage du serveur Plesk :

  1. Allez dans Outils & Paramètres > Gestionnaire de sauvegardes et cliquez sur la sauvegarde que vous voulez déplacer.

    Remarque : les sauvegardes stockées dans le stockage FTP ont une icône correspondante dans la colonne Notes du système.

  2. Cliquez sur Copier vers le stockage du serveur.
