Servir les clients non techniques

Certains de vos clients peuvent manquer de connaissances techniques en matière d'administration système, il est donc naturel qu'ils se sentent perdus à la vue de l'ensemble des outils offerts par le Panneau Client. Ils risquent même de corrompre les fichiers de configuration et de compliquer la résolution des problèmes s'ils essaient d'utiliser ces outils. Pour éviter ce genre de situation et présenter à vos clients un Panneau Client simple et convivial, nous vous conseillons de les abonner au pack d'hébergement Simple par défaut (Default Simple) disponible dans Packs de services > onglet Packs d'hébergement.

Ce pack permet aux clients d'exécuter uniquement des opérations de routine toutes simples. Quant aux tâches plus complexes, ils peuvent les laisser à votre service d'assistance. Même s'il ne contient que les outils les plus populaires et les plus fréquemment utilisés, vous êtes libre d'adapter les paramètres du pack.

Si vous n'avez pas l'intention de créer des packs spécifiques pour un tel public, il est possible de limiter le nombre d'outils et de paramètres sur la base par abonnement à l'aide des instructions que nous indiquons dans cette section. Les instructions suivantes expliquent comment simplifier l'interface utilisateur : en pratique, vous devez masquer les ressources et outils que vos clients n'utiliseront pas.

Masquage des ressources inutilisées

Le principe est simple : dans les paramètres d'abonnement ou du pack d'hébergement, dans l'onglet Ressources, définissez toutes les limites inutilisées sur 0. En fonction de la Visibilité des fonctions d'hébergement dans le Panneau Client, les ressources de ce type seront masquées dans le Panneau Client.

Par exemple, si vous configurez la limite des boîtes mail sur 0, la section Mail ne s'affichera pas dans le Panneau Client. Pour en savoir plus sur les ressources, reportez-vous à la section Ressources.

Masquage des outils inutilisés

Le jeu d'outils disponibles pour les clients dans le Panneau Client est défini par les paramètres des packs ou des abonnements, dans l'onglet Droits. Par exemple, si vous décochez l'option Gestion de la zone DNS, les outils pour gérer le DNS ne s'afficheront pas dans le Panneau Client. Pour en savoir plus sur les droits, consultez la section Droits.

Ressources et outils du pack d'hébergement simple par défaut

Initialement, le pack d'hébergement Simple par défaut inclut les outils et les paramètres suivants :

  • Accès à l'hébergement Web
  • Presence Builder
  • Gestionnaire de fichiers
  • Statistiques des sites Web
  • Protection des sites avec SSL/TLS
  • Applications
  • Bases de données
