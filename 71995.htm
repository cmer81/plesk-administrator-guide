Web Server IIS (Windows)

Plesk pour Windows pour utilise le serveur HTTP IIS (http://www.iis.net/) pour héberger et gérer les sites Web.

Configurer le serveur Web par défaut

La configuration IIS par défaut est définie par l'hébergeur à l'aide des outils IIS comme le gestionnaire IIS. Cette configuration par défaut est appliquée à tous les sites Web sur le serveur. Toutefois, un certain nombre de paramètres de configuration peuvent être modifiés pour chaque site Web dans l'interface de Plesk.

Personnaliser la configuration du serveur Web

Les propriétaires de sites Web peuvent avoir besoin de capacités de serveur Web personnalisées qui ne sont pas fournies avec la configuration par défaut. Par exemple, des types de fichiers index inhabituels ou des accès restreints au site par adresse IP.

Vous ou les propriétaires des sites Web pouvez configurer les paramètres du serveur Web pour un site Web en précisant les paramètres IIS dans le Panneau Client. La configuration personnalisée du site Web écrase la configuration par défaut. Pour en savoir plus sur la configuration personnalisée d'ISS, consultez la section Ajuster les paramètres IIS pour les sites Web.

Ajuster les paramètres IIS pour les sites Web

Vous ou les propriétaires des sites Web pouvez personnaliser la configuration IIS pour un site Web particulier dans le Panneau Client, dans Sites Web & Domaines > sélectionner un domaine > Paramètres du IIS. Tous les paramètres sont divisés en trois groupes :

  • Paramètres communs

    La section Paramètres communs contient les paramètres que les propriétaires de sites Web souhaitent généralement ajuster. Par exemple, pour ajouter des fichiers index personnalisés ou autoriser l'exploration des répertoires. Pour chaque paramètre, vous pouvez soit saisir une valeur personnalisée, soit utiliser la configuration IIS par défaut (en sélectionnant la valeur Par défaut).

  • Paramètres de sécurité du répertoire

    Les paramètres de la section Paramètres de sécurité du répertoire vous permettent d'appliquer les connexions HTTPS et d'interdire les accès anonymes au site.

  • Paramètres de restriction d'accès

    Les paramètres de la section Paramètres de restriction d'accès vous permettent de contrôler l'accès au site Web par adresse IP.

    Remarque : contrairement aux autres paramètres de serveur Web, le paramètre Refuser l'accès au site n'écrase pas la liste des adresses IP fournies par la configuration par défaut, car elle la complète. En cas de conflit (par exemple, si vous autorisez une adresse IP refusée par la configuration par défaut), vos valeurs seront utilisées.
