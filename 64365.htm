Se connecter aux dossiers Web sur OS X

Pour connecter un dossier Web en tant que lecteur réseau à votre ordinateur sous Mac OS X :

  1. Ouvrez le Finder.
  2. Cliquez sur Aller et sélectionnez l'option Se connecter au serveur.
  3. Indiquez l'emplacement de votre dossier partagé.

    Pour trouver l'emplacement de votre dossier Web, connectez-vous à Plesk, allez dans Partage de fichiers, trouvez le dossier requis et cliquez sur l'icône dans le coin supérieur droit du Gestionnaire de fichiers. Le lien vers le dossier Web actuel s'affichera dans la fenêtre ouverte.

  4. Cliquez sur Se connecter.
  5. Indiquez le nom d'utilisateur et le mot de passe que vous utilisez pour vous connecter à Plesk.
  6. Sélectionnez l'option Enregistrer ce mot de passe dans mon trousseau. Ainsi, vous n'aurez plus à le saisir à chaque connexion au serveur.
  7. Cliquez sur OK.

    Si vous utilisez SSL/TLS, il se peut que vous receviez une alerte vous indiquant que le certificat SSL/TLS sur le serveur n'est pas valide. Dans ce cas, cliquez pour passer l'avertissement SSL/TLS.

    Une fois que le lecteur réseau est monté, un raccourci vers ce lecteur sera placé sur votre bureau.

  8. Ouvrez Préférences système et sélectionnez Comptes dans le menu Présentation.
  9. Sélectionnez votre compte utilisateur, puis cliquez l'onglet Élément de démarrage.
  10. Sélectionnez le raccourci du lecteur réseau sur votre bureau et faites-le glisser vers la fenêtre Élément de démarrage (dans Préférences système > Comptes).
  11. Si vous ne voulez pas que le lecteur monté s'ouvre automatiquement dans le Finder à chaque fois que vous démarrez le Mac, cochez la case Masquer correspondante.
  12. Cliquez sur OK.
