Utiliser des thèmes personnalisés

Vous pouvez personnaliser l'interface de Plesk en changeant l'image du logo, l'URL associée et le texte de la barre de titre du navigateur. Plesk Onyx vous offre aussi la possibilité de modifier le "style Plesk" (aspect visuel et marque) à l'aide de thèmes personnalisés.

Le thème Plesk est un jeu complet d'images et de styles CSS définissant l'aspect visuel de l'interface utilisateur Plesk. Vous ne pouvez pas changer de thème mais vous pouvez personnaliser l'apparence de Plesk en redéfinissant des styles et images du thème (sans substitution physique), ce qui permet d'annuler facilement les modifications.

Un thème avec un jeu redéfini de styles et images est appelé thème personnalisé.

Plesk fournit un pack de thème par défaut pour créer votre thème personnalisé. Ce pack n'inclut pas tous les styles et images. Par exemple, le fichier CSS est vide au départ dans le pack de thème par défaut. Vous pouvez y ajouter les définitions de styles. Vous ne pouvez modifier que les styles correspondants aux éléments "décoratifs" comme les couleurs, les arrières-plans et les polices. Vous pouvez bien entendu ajouter des styles qui redéfinissent la mise en page. Toutefois, nous vous le déconseillons fortement, car certaines zones ou fonctions de Plesk risquent de devenir inopérantes.

Ces thèmes personnalisés peuvent être créés et appliqués à Plesk par les administrateurs serveurs et les revendeurs ; un thème par administrateur serveur ou par compte revendeur. Les modifications de l'apparence de Plesk appliquées par l'administrateur serveur sont visibles pour tous les utilisateurs de Plesk. Les modifications appliquées par un revendeur sont visibles uniquement par les revendeurs et leurs utilisateurs.

Pour préparer et appliquer un thème personnalisé, vous devez procéder comme suit :

  1. Récupérez les fichiers qui serviront de base pour créer votre thème personnalisé. Pour cela, connectez-vous à votre serveur via SSH (Linux) ou via une connexion Remote Desktop (Windows) et exécutez l'utilitaire de ligne de commande branding_theme (sur Windows il peut s'appeler branding_theme.exe). Pour en savoir plus, consultez la section Obtenir un pack de thème de Plesk.

    Remarque : si vous accédez à Plesk depuis un niveau revendeur et si vous n'avez pas de droit d'accès administrateur au serveur via SSH ou Remote Desktop, contactez votre hébergeur pour obtenir de l'aide pour votre pack.

  2. Téléchargez le dossier archive, dézippez-le et modifiez les fichiers selon vos besoins : ajoutez les styles et remplacez les fichiers graphiques par les vôtres. Veillez à préserver l'intégrité de la structure du répertoire. Pour en savoir plus, consultez la section Modifier les fichiers.
  3. Créez un dossier ZIP avec le répertoire du thème modifié. Envoyez le fichier archive sur le serveur et exécutez l'utilitaire de ligne de commande branding_theme pour appliquer le thème personnalisé. Pour en savoir plus, consultez la section Installer des thèmes dans Plesk.

Remarque : le nouveau thème sera appliqué une fois que vous vous serez déconnecté puis reconnecté à Plesk.

Si vous voulez restaurer les styles et images par défaut de Plesk, supprimez le thème personnalisé. Pour en savoir plus, consultez la section Supprimer des thèmes de Plesk.

Vous pouvez trouver la description de toutes les commandes et options de l'utilitaire branding_theme dans le guide sur la CLI Linux ou le guide sur la CLI Windows.
