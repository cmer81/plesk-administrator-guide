Modifier votre mot de passe et vos coordonnées

Pour changer votre mot de passe :

  1. Cliquez sur le lien Changer le mot de passe dans le volet de navigation.
  2. Entrez votre nouveau et votre ancien mot de passe.
  3. Cliquez sur OK.

Pour mettre à jour vos coordonnées :

  1. Cliquez sur le lien Profils & Préférences dans le volet de navigation.
  2. Mettez vos données à jour puis cliquez sur OK.

Si vous avez oublié votre mot de passe :

  1. Dans la barre d'adresse de votre navigateur, saisissez l'URL de votre Plesk.

    Exemple : https://votre-serveur.com:8443.

  2. Appuyez sur la touche ENTRÉE. L'écran de connexion de Plesk s'affiche.
  3. Cliquez sur le lien Mot de passe oublié ?.
  4. Vous êtes invité à saisir votre nom d'utilisateur et votre adresse mail enregistrés dans le système. Saisissez votre nom d'utilisateur dans le champ Nom d'utilisateur, votre adresse mail telle qu'enregistrée sur le système dans le champ Adresse mail, puis cliquez sur OK.
  5. Si votre mot de passe ne peut être envoyé par mail car il était stocké par le système sous forme chiffrée, vous êtes invité à définir un nouveau mot de passe à l'aide d'un code secret généré à cette fin et envoyé à votre adresse mail.
  6. Une fois que vous avez reçu le mail du rappel de mot de passe, cliquez sur le lien dans le corps du message. Une nouvelle fenêtre du navigateur s'ouvre.
  7. À cette étape, indiquez votre nom d'utilisateur et un nouveau mot de passe.

    Le champ Code secret du formulaire doit être automatiquement rempli par le système. Si tel n'est pas le cas, copiez-le dans le presse-papiers à partir du message que vous avez reçu et collez-le dans le formulaire.

  8. Cliquez sur OK pour soumettre les données.

    Les instructions pour savoir comment restaurer votre mot de passe seront envoyées par mail à votre adresse.
