Ressources

Les Ressources (dans Packs de services > sélectionnez un pack > onglet Ressources) définissent les ressources système fournies avec l'abonnement.

Politique de sur-utilisation

Définit ce qui se passe si l'utilisation de l'espace disque et du trafic de l'abonnement atteint la limite :

  • La sur-utilisation n'est pas autorisée : l'abonnement est suspendu.
  • La sur-utilisation de l'espace disque et du trafic est autorisée : l'abonnement continue de fonctionner correctement si l'espace disque et le trafic sont sur-utilisés.

    Vous pouvez sélectionner l'option Me notifier par mail en cas de sur-utilisation pour envoyer des notifications aux utilisateurs dès que l'utilisation de l'espace disque ou du trafic dépasse les limites indiquées (options Notifier lorsque l'utilisation de l'espace disque max a été atteinte et Notifier lorsque l'utilisation du trafic max a été atteinte). Des notifications sont envoyées aux utilisateurs et/ou aux adresses mail indiqués dans Paramètres > Notifications : Limites d'utilisation des ressources dépassées par l'abonnement.

  • La sur-utilisation est autorisée : l'abonnement continue de fonctionner correctement quel que soit le type de ressources sur-utilisé.

    Vous pouvez sélectionner l'option Me notifier par mail en cas de sur-utilisation pour envoyer des notifications aux utilisateurs dès qu'une ressource est sur-utilisée. Des notifications sont envoyées aux utilisateurs et/ou aux adresses mail indiqués dans Paramètres > Notifications : Limites d'utilisation des ressources dépassées par l'abonnement.

Remarque : la politique de sur-utilisation ne s'applique pas aux limites définies sur le volume des boîtes mail. Par conséquent, même si vous activez la sur-utilisation, assurez-vous d'allouer assez d'espace disque aux boîtes mail.

Espace disque

Définit l'espace disque total alloué à l'abonnement. Il inclut l'espace disque occupé par tous les fichiers associés à votre abonnement : contenu de sites Web, bases de données, applications, boîtes mail, fichiers de log et de sauvegarde.

Notifier lorsque l'utilisation de l'espace disque max a été atteinte

Cette option est disponible uniquement si la sur-utilisation de l'espace disque et du trafic est autorisée. Cela active la limite de la notification pour sur-utilisation de l'espace disque. Lorsque le quota est atteint, Plesk envoie des notifications aux adresses mail des utilisateurs indiqués dans Paramètres > Notifications : Limites d'utilisation des ressources dépassées par l'abonnement.

Remarque : vous pouvez définir la limite de notification en Mo/Go/To ou en pourcentage. Si la limite doit être fixe, vous pouvez utiliser des Mo/Go/To (dans ce cas, quand vous avez ajouté un add-on qui améliore l'espace disque pour l'abonnement, cette limite ne change pas). Si la limite doit être flottante, vous pouvez saisir un pourcentage (dans ce cas, quand un add-on est ajouté, la limite est recalculée automatiquement). Par exemple : si votre espace disque = 10 Go, définissez la limite de notification sur 9 Go et ajouter un add-on avec 5 Go d'espace disque, alors la notification sera envoyée une fois la limite de 9 Go dépassée. Si vous définissez la limite des notifications sur 90 %, alors la notification est envoyée une fois que la limite de 13,5 Go est dépassée.

Trafic

Total des données pouvant être transférées à partir des sites Web de l'abonnement et parts FTP/Samba en un mois.

Notifier lorsque l'utilisation du trafic max a été atteinte

Cette option est disponible uniquement si la sur-utilisation de l'espace disque et du trafic est autorisée. Cela active la limite de la notification pour l'utilisation du trafic. Lorsque le quota est atteint, Plesk envoie des notifications aux adresses mail des utilisateurs indiqués dans Paramètres > Notifications : Limites d'utilisation des ressources dépassées par l'abonnement.

Sites publiés avec Presence Builder

Nombre de sites Web pouvant être publiés avec Presence Builder.

Domaines

Nombre total de noms de domaine que l'abonné pourra héberger dans l'abonnement. Sont inclus les sites Web, les configurations des redirections Web pointant vers les sites Web hébergés sur d'autres serveurs et les noms de domaine où il n'y a pas encore de configuration de site Web ou de redirection de site Web (domaines sans hébergement).

Sous-domaines

Nombre total de sous-domaines que l'abonné pourra héberger dans l'abonnement.

Alias de domaine

Nombre total d'alias de noms de domaine que l'abonné pourra utiliser pour ses sites Web.

Boîtes mail

Nombre total de boîtes mail que l'abonné peut héberger dans l'abonnement.

Volume de la boîte mail

Total max. d'espace disque à chaque boîte mail dans un abonnement pour stocker les mails et fichiers joints de réponses automatiques.

Quota total des boîtes mail (hébergement Windows)

Volume total d'espace disque total en mégaoctets disponible pour toutes les boîtes mail dans l'abonnement.

Listes de diffusion

Nombre total de listes de diffusion que l'abonné peut héberger dans l'abonnement.

Comptes FTP supplémentaires

Nombre maximal de comptes FTP utilisés pour accéder aux fichiers et dossiers créés dans un abonnement. Ce nombre n'inclut pas le compte qui est toujours créé lors de la création de l'abonnement.

Bases de données (hébergement Unix)

Nombre de bases de données pouvant être créées sur les serveurs de bases de données Plesk et utilisées par les sites Web de l'abonnement.

Bases de données MySQL et Bases de données de Microsoft SQL (hébergement Windows).

Nombre maximal de bases de données MySQL et Microsoft SQL Server pouvant être créées sur les serveurs de bases de données de Plesk et utilisées par les sites Web de l'abonnement.

Quota total des bases de données MySQL et Quota total de bases de données MS SQL (hébergement Windows)

Volume maximal d'espace disque que les bases de données MySQL et Microsoft SQL Server de l'abonnement peuvent occuper.

Taille du fichier de bases de données MS SQL (hébergement Windows)

Taille maximale du fichier d'une base de données Microsoft SQL Server. Remarque : si une base de données comporte plusieurs fichiers, elle peut occuper plus d'espace disque que vous ne l'avez indiqué ici.

Taille du fichier de log de la base de données MS SQL (hébergement Windows)

Taille maximale du fichier de log d'une base de données Microsoft SQL Server. Remarque : si une base de données comporte plusieurs fichiers de log, elle peut occuper plus d'espace disque que vous ne l'avez indiqué ici.

Période de validité / Date d'expiration

Période de validité d'un abonnement.

Dans les propriétés du pack de services, c'est la Période de validité : elle est utilisée uniquement pour les abonnements qui viennent d'être créés. La date d'expiration de l'abonnement dépend de cette période.

Dans les propriétés de l'abonnement, c'est la Date d'expiration : à cette date, l'abonnement sera suspendu. Autrement dit, tous les sites dans l'abonnement seront suspendus, leurs services Web, FTP et de messagerie ne seront plus disponibles pour les internautes. En outre, ni les abonnés ni leurs utilisateurs ne pourront plus se connecter au Panneau Client.

Les abonnements ne sont pas automatiquement renouvelés. Donc, pour que les services de l'abonnement refonctionnent, vous devrez activer l'abonnement manuellement.

Applications Java

Nombre total d'applications Java ou d'applets pouvant être hébergés sur les sites Web dans l'abonnement.

Utilisateurs Web

Nombre total des pages Web personnelles que l'abonné peut héberger pour d'autres utilisateurs dans ses sites Web. Ce service est surtout utilisé par des établissements scolaires qui hébergent les pages personnelles de leurs étudiants et de leur personnel. Ces pages ont généralement des adresses comme http://exemple.com/~webuser.

Connexions ODBC (hébergement Windows)

Nombre total de connexions ODBC que l'abonné peut utiliser dans l'abonnement.
