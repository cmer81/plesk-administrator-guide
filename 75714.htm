Volume total de l'espace disque qui peut être utilisé par les bases de données MySQL sur un abonnement

OLD_MYSQL_DATABASE_QUOTA

Uniquement les événements MySQL

Espace disque actuellement utilisé par les bases de données MySQL sur un abonnement

OLD_MYSQL_DATABASE_USAGE

Uniquement les événements MySQL

Volume total de l'espace disque qui peut être utilisé par les bases de données Microsoft SQL Server sur un abonnement

OLD_MICROSOFT_SQL_DATABASE_QUOTA

Uniquement pour les événements MS SQL Server

Espace disque actuellement utilisé par les bases de données Microsoft SQL Server sur un abonnement

OLD_MICROSOFT_SQL_DATABASE_USAGE

Uniquement pour les événements MS SQL Server

La limite douce peut être modifiée dans le fichier panel.ini en précisant l'option databaseManagement.diskSpaceSoftQuota. La valeur est le pourcentage de la limite (par défaut 85 %). Par exemple :

[databaseManagement]

diskSpaceSoftQuota=50

Exemple du gestionnaire d'événements :

C:\temp\mssql_soft.bat <NEW_DOMAIN_NAME> <NEW_DOMAIN_GUID> <NEW_CLIENT_GUID> <NEW_MICROSOFT_SQL_DATABASE_QUOTA> <NEW_MICROSOFT_SQL_DATABASE_USAGE> <OLD_DOMAIN_NAME> <OLD_DOMAIN_GUID> <OLD_CLIENT_GUID>

echo "--------------" >> c:\temp\mssql_soft_handler.log

date /T >> c:\temp\mssql_soft_handler.log

echo "NEW_DOMAIN_NAME: %1" >> c:\temp\mssql_soft_handler.log

echo "NEW_DOMAIN_GUID: %2" >> c:\temp\mssql_soft_handler.log

echo "NEW_CLIENT_GUID: %3" >> c:\temp\mssql_soft_handler.log

echo "NEW_MICROSOFT_SQL_DATABASE_QUOTA: %4" >> c:\temp\mssql_soft_handler.log

echo "NEW_MICROSOFT_SQL_DATABASE_USAGE: %5" >> c:\temp\mssql_soft_handler.log

echo "OLD_DOMAIN_NAME: %6" >> c:\temp\mssql_soft_handler.log

echo "OLD_DOMAIN_GUID: %7" >> c:\temp\mssql_soft_handler.log

echo "OLD_CLIENT_GUID: %8" >> c:\temp\mssql_soft_handler.log

echo "--------------" >> c:\temp\mssql_soft_handler.log
