Droits

Les privilèges spécifiques aux revendeurs (ceux qui n'affectent pas les abonnements de services qu'ils créent pour leurs clients) sont les suivants :

Possibilité d'utiliser l'API distante.

Définit si un revendeur peut gérer des sites Web à distance à l'aide d'applications personnalisées. L'API distante est une interface qui peut être utilisée pour développer des applications personnalisées intégrées à des sites Web. Ces applications peuvent être utilisées pour automatiser la configuration de comptes d'hébergement et le provisioning de services pour les clients achetant des services d'hébergement sur votre site. Pour en savoir plus, consultez la documentation sur l'API de Plesk.

Accès à Plesk

Définit si un revendeur peut utiliser l'interface utilisateur graphique de Plesk.

Création d'un compte client

Définit si un revendeur peut créer des comptes utilisateur et des abonnements pour ses clients dans Plesk.

Possibilité de survente

Définit si un revendeur peut configurer une politique de survente, en d'autres termes, s'il peut lui-même définir si la survente est autorisée ou non.

Autres droits

Remarque : les autres droits sont les mêmes que dans les abonnements de services.
