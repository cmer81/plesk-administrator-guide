Rapports techniques de Plesk

Si vous deviez rencontrer des problèmes avec Plesk ou l'un de ses services, vous aurez peut-être besoin de contacter l'assistance technique pour les résoudre. Généralement, le dépannage requiert des informations contenues dans l'un des nombreux fichiers de log du serveur. Si vous ne fournissez pas ces informations lors de la création du ticket d'assistance, l'équipe d'assistance vous contactera pour vous demander certains fichiers de log pour résoudre le problème. Cet échange d'informations peut entraîner des délais significatifs et avoir un impact sur les niveaux de services ainsi que sur la satisfaction du client.

Pour éviter ces délais, vous pouvez générer un rapport technique avant même de contacter l'assistance. Ce rapport technique recueille tous les fichiers de log et les autres informations pertinentes pour résoudre les problèmes. Les informations et fichiers sont téléchargés sur les serveurs Plesk. Ainsi, l'équipe technique dispose de toutes les informations requises dès le début, ce qui diminue considérablement les délais de résolution des problèmes.

Remarque : le rapport technique ne contient aucune donnée privée, uniquement des informations techniques.

Pour générer un rapport technique, allez sous Outils & Paramètres > Assistance (groupe Plesk). Sélectionnez Générez un rapport technique avant de contacter l'assistance. (recommandé) et cliquez sur Suivant. Une fois le rapport généré, cliquez sur Envoyer le rapport. Vous pouvez afficher le contenu du rapport avant de l'envoyer. Pour cela, cliquez sur Télécharger. Une fois le rapport envoyé sur les serveurs Plesk, vous recevez l'ID unique de votre rapport technique. Cliquez sur Contacter l'assistance pour envoyer un ticket d'assistance et assurez-vous d'indiquer l'ID du rapport lorsque vous décrivez votre problème.

Si vous devez générer un rapport technique alors que le panneau de contrôle Plesk est indisponible, vous pouvez le générer via la ligne de commande. Connectez-vous au serveur via SHH ou RDP et exécutez les commandes suivantes :

# plesk bin report --collect

Cette commande génère le rapport et le place dans le répertoire /usr/local/psa/var/reports/collected/. Vous pouvez vérifier le contenu du rapport si vous le voulez. Pour envoyer le rapport à l'assistance, exécutez la commande suivante :

# plesk bin report --send=<nom du rapport généré>

Si vous ne voulez pas voir le rapport avant de l'envoyer, vous pouvez exécuter la commande suivante :

# plesk bin report --send

Cette commande génère et envoie immédiatement le rapport.
