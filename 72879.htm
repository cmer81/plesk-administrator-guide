Modifier les identifiants d'accès de l'administrateur de la base de données

En général, pour modifier les identifiants d'accès de Plesk à un serveur de bases de données, vous devez d'abord modifier ces identifiants sur le serveur de bases de données. Ensuite, vous devez les mettre à jour dans Plesk. Hormis pour les situations décrites ci-dessous, vous ne pouvez pas modifier les identifiants depuis Plesk.

Remarque : généralement, lorsque ces identifiants sont mis à jour dans Plesk, la modification n'est pas répercutée sur le serveur de bases de données.

Première exception, Plesk pour Windows : vous pouvez modifier le mot de passe de l'administrateur de la base de données. Deuxième exception, PostgreSQL : la modification des identifiants de l'administrateur de la base de données effectuée dans Plesk est répercutée dans PostgreSQL.

Mettre à jour les identifiants d'accès de l'administrateur de la base de données dans Plesk

Si vous avez changé les identifiants de l'administrateur sur le serveur de bases de données, vous devez également les changer dans Plesk.

Pour mettre à jour les identifiants de l'administrateur du serveur de bases de données dans Plesk (Linux) :

  1. Allez dans Outils & Paramètres > Serveurs de bases de données.
  2. Cliquez sur le nom d'hôte d'un serveur de bases de données.
  3. Indiquez le nouveau nom d'utilisateur et le nouveau mot de passe de l'administrateur du serveur de bases de données.

Pour mettre à jour les identifiants de l'administrateur du serveur de bases de données dans Plesk (Windows) :

  1. Allez dans Outils & Paramètres > Serveurs de bases de données.
  2. Cliquez sur le nom d'hôte du serveur de bases de données, puis sur Paramètres.
  3. Indiquez le nouveau nom d'utilisateur et le nouveau mot de passe de l'administrateur du serveur de bases de données.

Modifier le mot de passe de l'administrateur de la base de données (Windows)

Vous pouvez modifier le mot de passe de l'utilisateur actuellement utilisé par Plesk pour accéder au serveur de bases de données. Quand vous modifiez le mot de passe dans l'interface de Plesk, il est automatiquement changé sur le serveur de bases de données et mis à jour dans les paramètres du serveur de bases de données dans Plesk.

Pour changer le mot de passe de l'administrateur du serveur de bases de données :

  1. Allez dans Outils & Paramètres > Serveurs de bases de données.
  2. Cliquez sur le nom d'hôte d'un serveur de bases de données.
  3. Cliquez sur Modifier le mot de passe.
  4. Saisissez le nouveau mot de passe, puis cliquez sur OK.
