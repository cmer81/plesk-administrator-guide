Serveurs de bases de données

Plesk a besoin de serveurs de bases de données afin de stocker ses propres bases de données, c'est-à-dire, les bases de données utilisées par ses composants (par ex. pour le service de messagerie Web) ainsi que les bases de données créées par l'hébergement des sites Web des clients et des applications Web (par ex. une base de données WordPress).

Plesk est compatible avec la plupart des moteurs de bases de données. Par ailleurs, Plesk est fourni avec les outils de gestion de bases de données correspondants. Pour fonctionner, Plesk peut utiliser des serveurs de bases de données situées sur le même serveur ou sur une machine distante.

Par défaut, sur les serveurs Linux, les moteurs de bases de données sont installés sur le même serveur que Plesk, depuis le répertoire de votre fournisseur d'OS (Linux) ou votre répertoire personnalisé. Sur les serveurs Windows, les moteurs de bases de données sont dans le même pack que Plesk et installés sur le même serveur que Plesk.

Logiciel de serveur de bases de données dans Plesk pour Linux

Plesk Onyx pour Linux prend en charge les serveurs de bases de données MySQL et PostgreSQL. Plesk utilise des packs MySQL et PostgreSQL depuis le répertoire de votre système d'exploitation. MySQL 5.5 constitue une exception. Fourni dans le même pack que Plesk, MySQL 5.5 n'est compatible qu'avec un nombre restreint de systèmes d'exploitation.

MySQL est installé pendant l'installation de Plesk. Vous pouvez installer PostgreSQL en plus via Outils & Paramètres > Mises à jour et à niveau > Ajouter/Supprimer des composants > Serveur PostgreSQL.

Vous pouvez également enregistrer des serveurs de bases de données externes MySQL et PostgreSQL dans Plesk. Pour en savoir plus, consultez la section Ajouter et supprimer des serveurs de bases de données. La liste des versions de serveurs de bases de données prises en charge est fournie dans les Notes de mises à jour .

Sur les installations Plesk CloudLinux, vous pouvez utiliser MySQL 5.5. MySQL 5.5 est fourni dans le pack avec Plesk. Vous pouvez l'installer via Mises à jour et à niveau > Ajouter/Supprimer des composants.

Logiciel de Serveur de bases de données dans Plesk pour Windows

Plesk Onyx pour Windows prend en charge Microsoft SQL Server Express 2008 et 2012, et MySQL 5.1. Par défaut, MySQL et Microsoft SQL Server 2008 sont installés pendant l'installation de Plesk. Vous pouvez installer Microsoft SQL Server 2012 en plus via Outils & Paramètres > Mises à jour et à niveau > Ajouter/Supprimer des composants.

Vous pouvez également enregistrer des serveurs externes Microsoft SQL Server et MySQL dans Plesk. Pour en savoir plus, consultez la section Ajouter et supprimer des serveurs de bases de données.

Plesk pour Windows utilise ODBC pour fonctionner avec d'autres moteurs de bases de données. Pour en savoir plus, consultez la section Se connecter à des bases de données externes (Windows).

La liste des versions de serveurs de bases de données prises en charge est fournie dans les Notes de mises à jour .

 

Remarque : les instructions relatives à la gestion des bases de données dans Plesk sont disponibles dans la section Gérer les bases de données du site Web.
