Statistiques et rapports

Plesk enregistre les statistiques et fournit des rapports sur les dépassements de limites de chaque objet Plesk : boîte mail, domaine ou abonnement. Les statistiques sont stockées pour un mois mais tous les rapports sont créés pour la période rapport, c'est-à-dire une période au courant du dernier mois (par exemple, un ou deux jours, une semaine, etc). Vous pouvez configurer cet intervalle sous Outils & Paramètres > Paramètres du serveur de messagerie > Activer les limites pour les mails sortants > Afficher les rapports basés sur les statistiques pour les derniers / les dernières [période de temps].

Afin de consulter les rapports sur les tentatives de dépassement de limite des mails sortants :

Allez dans Outils & Paramètres > Contrôle des mails sortants et sélectionnez l'onglet avec les informations dont vous avez besoin :

  • Pour voir un récapitulatif des informations, allez dans l'onglet Général. Vous y trouverez le nombre d'objets Plesk qui ont tenté de dépasser les limites ainsi que la liste de ces objets.
  • Les objets qui ne dépassent pas les limitent ont le statut de messagerie Envoi. Si un objet a atteint ses limites, le statut de son mail change en Envoi non lancé.

    Remarque : si Plesk est installé avec SmarterMail, les limites des mails sortants sont définies du côté de SmarterMail à l'aide de l'API SmarterMail. Dans ce cas, aucun filtrage n'est effectué par Plesk. Par conséquent, le statut des objets de Plesk n'est pas disponible.

  • Si vous avez besoin de plus d'informations au sujet d'un objet Plesk particulier, allez dans l'onglet correspondant (Adresses mail, Domaines ou Abonnements). Cliquez sur l'objet concerné et consultez le rapport Tentative d'envoi de mail une fois la limite atteinte.

    De plus, dans la section Mails envoyés, les statistiques quotidiennes par heure sont indiquées pour un objet Plesk : vous pouvez voir le nombre de tentatives de dépassement par heure pour un jour spécifique.

Remarque : le lien Contrôle des mails sortants s'affiche uniquement si les limites sont activées dans les Paramètres du serveur de messagerie .
