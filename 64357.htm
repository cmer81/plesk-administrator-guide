Accéder aux fichiers et les utiliser

Il y a deux manières d'utiliser les fichiers partagés :

  • Si vous avez besoin d'utiliser fréquemment les fichiers, connectez un dossier Web sur le serveur à votre ordinateur.
  • Si vous avez besoin de temps en temps d'accéder aux fichiers, utilisez l'interface de Plesk (section Fichiers).

Pour utiliser le Gestionnaire de fichiers intégré à votre système d'informations afin de travailler avec ces fichiers :

  1. Dans Plesk, cliquez sur la section Fichiers. Le Gestionnaire de fichiers s'ouvre.
  2. Utilisez les icônes et liens suivants pour travailler avec les fichiers et répertoires.

    Dans la zone de gauche :

    • Envoyer les fichiers. Cette option démarre un assistant pour vous aider à envoyer des fichiers et des répertoires vers le serveur.
    • Fichiers personnels. Cette option vous dirige vers le répertoire privé dans lequel vous pouvez placer les fichiers auxquels vous seulement avez accès. Si vous devez utiliser de l'espace de stockage sur le serveur à des fins de sauvegarde ou si vous devez accéder à des fichiers via Internet, placez vos fichiers dans ce répertoire. Pour en savoir plus, consultez la section Envoyer vos fichiers vers un répertoire privé sur le serveur.
    • Fichiers partagés. Cette option vous dirige vers le répertoire partagé (ou commun) dans lequel vous devriez placer les fichiers qui doivent être disponibles pour les autres utilisateurs de votre organisation. Lorsque vous collaborez avec d'autres employés, placez les fichiers dans ce répertoire. Pour en savoir plus, consultez la section Partager les fichiers avec d'autres utilisateurs dans l'organisation.

    La zone à droite affiche une liste de fichiers et de répertoires situés dans le répertoire actuellement sélectionné, ainsi qu'une barre d'outils avec les éléments suivants :

    • Partager. Sélectionnez les fichiers depuis n'importe quel répertoire et cliquez sur cette option pour déplacer les fichiers vers le Dossier partagé. Cela les rendra disponibles à tous les utilisateurs autorisés dans votre système d'informations.
    • Envoyer par mail un lien. Sélectionnez des fichiers dans n'importe quel répertoire et cliquez sur cette option pour envoyer un mail afin de notifier aux utilisateurs l'emplacement des fichiers, de sorte qu'ils puissent les télécharger.
    • Supprimer. Sélectionnez les fichiers et cliquez sur cette option pour les supprimer définitivement.
    • Plus. Ce menu vous permet d'accéder aux opérations suivantes : créer un dossier, copier ou déplacer des fichiers et des dossiers.
    • Icône (Lien vers ce dossier). Cliquez sur cette icône pour afficher le presse-papiers ou y copier les adresses Internet du dossier actuel.
